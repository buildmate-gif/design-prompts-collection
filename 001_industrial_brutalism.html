// ABOUT THE SYSTEM

Structural
Philosophy

ブルータリズムは1950年代の建築運動に端を発し、装飾的な要素を排除し、 素材そのものの質感と構造的な誠実さを前面に押し出す思想である。 コンクリート、鉄骨、レンガ——これらは隠されるのではなく、むしろ誇示される。

Webデザインにおいても同様のアプローチが可能だ。 グリッドシステムは地図のように可視化され、ボーダーは壁として機能し、 システムの骨格は利用者の目に直接さらされる。

Core Functions

// 03 PRINCIPLES DEFINED
01

Exposed Logic

裏側に隠されるべき配管や構造材をあえて露出させることで、 システムの動作原理を視覚化します。要素の境界線は曖昧にせず、 明確な区切りとして機能します。コードもデザインも、 内部構造を隠すことは誠実さの欠如とみなされます。

02

Anti Design

ユーザーに媚びない無骨なインターフェース。 過度な余白やラウンドコーナーを排除。 見た目の美しさより、機能の明快さを優先する。

03

Function First

すべてのアクションは直接的でなければなりません。 機能美の極限を目指した設計思想。 ボタンはボタンに見え、リンクはリンクに見える。
