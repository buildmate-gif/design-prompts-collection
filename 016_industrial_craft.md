# Pattern 01 — "Industrial Craft"
**テーマ：工業×職人美 / 建設・製造業・技術系サービスに最適**

```
【UI/UXデザイン要件 — Industrial Craft】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0. 出力形式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 出力：HTML + CSS（単一ファイル）または JSX + Tailwind CSS（単一ファイル）
- 外部フォント：Google Fonts（Noto Sans JP × Space Grotesk × DM Mono）のみ許可
- JavaScript：控えめなマイクロインタラクション（CSS優先、JS補助）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 参照スタイル・世界観
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 参照誌：Wallpaper* / Domus / Mark Magazine
- キーワード：Brutalist Function / Raw Material / Precision Craft
- 一言で表すなら「工場の設計図が、そのままWebになったような無駄のなさ」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. カラーパレット（厳守）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- ベース：コンクリートグレー #D9D6CF / スモークホワイト #EDEAE4
- テキスト：インクブラック #1C1B18 / スレート #5C5C5C
- アクセント：1色のみ（インダストリアルオレンジ #C45C1A）
- 禁止：純白 #FFFFFF、純黒 #000000、パステル系、グロッシーな質感

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. タイポグラフィ（階層定義）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 見出し（H1）：Space Grotesk / font-size: clamp(3rem, 7vw, 6rem)
  font-weight: 700 / letter-spacing: -0.04em / uppercase 必須
- 見出し（H2）：Space Grotesk / font-size: clamp(1.5rem, 3vw, 2.5rem)
  font-weight: 500 / letter-spacing: -0.02em
- 本文（body）：Noto Sans JP / font-size: 1rem / line-height: 1.8
- 数値・コード的要素：DM Mono / letter-spacing: 0.05em（スペック表示などに）
- モノスペースとグロテスクの意図的な混在で「仕様書の緊張感」を演出すること

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. レイアウト原則
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- グリッド：12カラムベース（均等配分禁止）
  例：[8col コンテンツ] + [4col メタ情報] / [2col 番号] + [10col 見出し]
- 左端に固定の「番号カラム」（01 / 02…）を設け、スクロールに追従させること
- Whitespace：section の padding は最低 6rem（モバイル 3rem）
- テキストブロックに水平方向の「目盛り線」を入れること（1px solid #1C1B1815）
- コンテンツ幅は最大 1100px、余白は非対称（左 5%、右 12% 程度）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 装飾ルール（機能美の美学）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【使用許可】
- 1px 極細ボーダー（border: 1px solid #1C1B1820）
- SVG feTurbulence によるコンクリート質感（opacity 0.04〜0.06）
- テキストアニメーション：opacity + translateY（0.5s ease-out、staggered）
- ホバー：アクセントカラーの細いアンダーライン展開（transform: scaleX）
- 数値カウントアップアニメーション（実績・規模感の表示に）

【完全禁止】
- 絵文字・アイコン一切禁止
- border-radius 8px 超（角丸禁止）
- グラデーション背景（配色の深みは明度差のみで表現）
- カード型のボックスシャドウ
- 均等割りの3カラムレイアウト
- フラットデザイン的な「色面」の多用

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. 「忘れられないディテール」の要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
以下のうち最低2つを必ず実装すること：
- [ ] 左端固定の縦方向ページ番号ナビ（01〜0N、スクロール連動でアクティブ表示）
- [ ] DM Monoで表示するスペック数値（1,200現場 / 15年 / 200社 等）の大型配置
- [ ] テキストの一部を uppercase + letter-spacing 拡大で「刻印」のように表示
- [ ] SVGラインによる「設計図的」な細い区切り（破線 or ドット）
- [ ] スクロール連動の水平移動テキスト（marquee的、CSSのみ）
```
