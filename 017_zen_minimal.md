# Pattern 02 — "Zen Minimal"
**テーマ：余白の哲学 / コンサル・士業・個人ブランディングに最適**

```
【UI/UXデザイン要件 — Zen Minimal】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0. 出力形式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 出力：HTML + CSS（単一ファイル）または JSX + Tailwind CSS（単一ファイル）
- 外部フォント：Google Fonts（Noto Serif JP × Cormorant Garamond）のみ許可
- JavaScript：一切使用しない（CSS Animationのみで完結させること）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 参照スタイル・世界観
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 参照：無印良品 / 原研哉デザイン / 東京国立博物館の空間設計
- キーワード：Ma（間）/ Emptiness as Design / Wabi-Sabi Typography
- 一言で表すなら「何も置かない、という選択そのものがメッセージになる空間」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. カラーパレット（厳守）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- ベース：和紙白 #F2EFE8 / 霞グレー #E5E0D8
- テキスト：墨色 #1E1C18 / 薄墨 #7A7570
- アクセント：1色のみ（辰砂 #8B3A3A または鉄紺 #1F3A5F のどちらか文脈で選択）
- 原則：3色以内に収めること。アクセントは1ページに5回以下の使用に限る

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. タイポグラフィ（階層定義）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 見出し（H1）：Noto Serif JP / font-size: clamp(2rem, 5vw, 4rem)
  font-weight: 300 / letter-spacing: 0.15em（漢字の余白を最大限に活かす）
- 見出し（H2）：Cormorant Garamond / font-size: clamp(1.2rem, 2.5vw, 2rem)
  font-style: italic / letter-spacing: 0.08em
- 本文（body）：Noto Serif JP / font-size: 1rem / line-height: 2.2 / letter-spacing: 0.05em
- 縦書き要素：writing-mode: vertical-rl を副見出しや引用に必ず使用
- テキスト量は最小化。1セクションあたりの本文は100文字以内を目標とする

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. レイアウト原則
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- グリッド：コンテンツ幅は最大 680px（意図的な狭さで集中させる）
- 余白こそが主役：各セクションは最低 12rem の垂直余白を持つこと
- 要素数の制限：1スクロールビュー内に配置する要素は最大3つ
- 水平ラインは使わず、余白のみでセクションを分離すること
- テキストの配置は左揃えを基本とし、中央揃えは見出しのみ許可

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 装飾ルール（非装飾の美学）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【使用許可】
- 0.5px の極細ライン（color: #1E1C1815 のみ）
- フェードイン（opacity: 0 → 1 / transition: 1.2s ease）
- 縦書きテキストの静的配置
- 和紙質感のSVG grain（feTurbulence / opacity: 0.02〜0.03）

【完全禁止】
- あらゆる装飾・影・グラデーション
- アニメーションの多用（1ページに最大2種まで）
- 画像の多用（使用する場合は白黒 filter: grayscale(100%) 必須）
- border-radius の使用（0px 固定）
- ホバーエフェクトの派手な演出

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. 「忘れられないディテール」の要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
以下のうち最低2つを必ず実装すること：
- [ ] 縦書き（writing-mode: vertical-rl）の日本語副見出しを左端に固定配置
- [ ] 漢字1文字を clamp(8rem, 20vw, 16rem) で背景的に配置（opacity: 0.04）
- [ ] 読み込み時のテキスト出現：文字ごとに 0.05s ずつ遅延するフェードイン
- [ ] ページ最下部に「—— 余白について ——」のような禅問答的フッター
- [ ] 全要素のtransitionを 0.8s cubic-bezier(0.25, 0.46, 0.45, 0.94) で統一
```
