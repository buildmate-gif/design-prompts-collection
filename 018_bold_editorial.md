# Pattern 03 — "Bold Editorial"
**テーマ：衝撃的な存在感 / ブランド立ち上げ・キャンペーン・メディアに最適**

```
【UI/UXデザイン要件 — Bold Editorial】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0. 出力形式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 出力：HTML + CSS（単一ファイル）または JSX + Tailwind CSS（単一ファイル）
- 外部フォント：Google Fonts（Noto Sans JP × Bebas Neue × DM Sans）のみ許可
- JavaScript：スクロール連動のみ許可（IntersectionObserver）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 参照スタイル・世界観
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 参照誌：i-D Magazine / Dazed / Numero Tokyo
- キーワード：Typographic Impact / Editorial Violence / Controlled Chaos
- 一言で表すなら「見た瞬間に動きが止まる。タイポグラフィが凶器になる瞬間」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. カラーパレット（厳守）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 以下の3パターンから1つ選択して使用すること：

  [A. アッシュ×シャープイエロー]
  ベース：#EDEAE2 / テキスト：#1A1A16 / アクセント：#D4A017

  [B. ディープグリーン×クリーム]
  ベース：#F0EDE6 / テキスト：#0F2218 / アクセント：#2D6A4F

  [C. チャコール×ダスティローズ]
  ベース：#EAE7E2 / テキスト：#1C1816 / アクセント：#B05A6A

- 禁止：純白、純黒、原色、彩度50%超えの色

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. タイポグラフィ（階層定義）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 見出し（H1）：Bebas Neue / font-size: clamp(5rem, 15vw, 14rem)
  letter-spacing: -0.02em / uppercase / 行をまたいで折り返しを許容する
- 見出し（H2）：DM Sans / font-size: clamp(1rem, 2vw, 1.5rem)
  font-weight: 300 / letter-spacing: 0.2em / uppercase（H1との対比を最大化）
- 本文（body）：Noto Sans JP / font-size: 0.95rem / line-height: 1.9
- 意図的な「文字サイズの極端な対比」を設計の核に置くこと
  （例：H1が14rem、直下のキャプションが0.7rem）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. レイアウト原則
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- グリッド：フルブリード（幅 100vw）を基本とし、コンテンツのみ最大 1200px
- H1テキストは画面を突き破るように配置（overflow: hidden 禁止）
- 画像とテキストを大胆に重ねること（z-index を積極的に使用）
- セクションの高さ：最低 100vh / テキストのみのセクションは 60vh
- 意図的な「余白の偏り」：パディングは左右で2倍以上の差を設けること

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 装飾ルール（過剰の美学）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【使用許可】
- テキストの clip-path / mix-blend-mode: multiply
- 斜め配置（transform: rotate(-2deg 〜 2deg））：微細な傾き
- SVGによる手書き的なラインやスクリブル（ページに1つ）
- スクロール速度差による視差（parallax）
- カーソルカスタマイズ（cursor: none + カスタムSVG）

【完全禁止】
- 丸みのある要素（border-radius: 0 固定）
- グラデーション背景
- ドロップシャドウ
- 3カラム均等配置
- 「優しい・親しみやすい」印象を与えるデザイン要素全般

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. 「忘れられないディテール」の要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
以下のうち最低2つを必ず実装すること：
- [ ] 画面幅を完全に超える大型見出しの配置（clip不要、はみ出し許容）
- [ ] テキストに mix-blend-mode: multiply を使い画像と溶け込む表現
- [ ] スクロールに反応して傾きが変わる要素（transform: rotate）
- [ ] カーソル追従のカスタムサークル（JS可、半径24px / mix-blend-mode: difference）
- [ ] 縦スクロールに連動する横スライド（translateX）セクション
```
