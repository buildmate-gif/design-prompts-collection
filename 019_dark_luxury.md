# Pattern 04 — "Dark Luxury"
**テーマ：夜の上質感 / ハイエンドサービス・プレミアム訴求・ナイトモードに最適**

```
【UI/UXデザイン要件 — Dark Luxury】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0. 出力形式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 出力：HTML + CSS（単一ファイル）または JSX + Tailwind CSS（単一ファイル）
- 外部フォント：Google Fonts（Noto Serif JP × Cormorant Garamond × DM Sans）
- JavaScript：スクロール連動・カーソル追従のみ許可

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 参照スタイル・世界観
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 参照：Bottega Veneta / Aesop / The Ritz-Carlton のブランドサイト
- キーワード：Nocturnal Elegance / Material Richness / Restrained Glamour
- 一言で表すなら「深夜のホテルのバー。照明が落ちていても品格がにじみ出る空間」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. カラーパレット（厳守）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- ベース：深夜色 #0F0E0C / ミッドナイト #1A1916
- テキスト：クリームホワイト #E8E4DC / フォグ #8A8680
- アクセント：1色のみ（ゴールド #B8964E または シルバーグレー #9AAAB4 から選択）
- 禁止：純白 #FFFFFF、鮮やかな発光色、蛍光系、青白いLED的カラー

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. タイポグラフィ（階層定義）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 見出し（H1）：Cormorant Garamond / font-size: clamp(2.5rem, 5vw, 4.5rem)
  font-weight: 300 / font-style: italic / letter-spacing: 0.04em
- 見出し（H2）：DM Sans / font-size: clamp(0.7rem, 1.2vw, 1rem)
  font-weight: 400 / letter-spacing: 0.2em / uppercase（極度に小さく）
- 本文（body）：Noto Serif JP / font-size: 0.95rem / line-height: 2.0
  color: #8A8680（本文はあえて薄くし、見出しのイタリックを際立たせる）
- ゴールドアクセントの文字：特定キーワードのみ color: #B8964E で差別化

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. レイアウト原則
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- グリッド：左寄りの非対称（コンテンツ幅 max 900px、左マージン 8%）
- セクションの高さ：100vh 基本、コンテンツは垂直方向で下1/3に配置
- 画像：全て grayscale(60%) + contrast(110%) で統一し、暗部に溶け込ませる
- オーバーレイ：画像の上に rgba(15,14,12,0.4) のカラーオーバーレイ必須
- 空間の「重力」を下方向に感じさせるレイアウト（上部に余白を多く置く）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 装飾ルール（陰影の美学）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【使用許可】
- 内側に光るような border: 1px solid rgba(184,150,78,0.2)（ゴールド選択時）
- backdrop-filter: blur(20px) + background: rgba(15,14,12,0.7) でグラスモーフ
- テキストのゆっくりしたフェードイン（opacity / 1.5s ease）
- ホバー時：テキストカラーが #8A8680 → #E8E4DC に遷移（0.4s）
- SVG grain texture（feTurbulence / opacity: 0.03）でフィルム的質感

【完全禁止】
- 白い背景・明るいセクション
- ビビッドカラーのアクセント
- box-shadow の白系 glow 効果
- border-radius 4px 超（エッジはシャープに保つ）
- ポップなアニメーション（バウンス・スケールアップ等）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. 「忘れられないディテール」の要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
以下のうち最低2つを必ず実装すること：
- [ ] 背景に極めて薄いノイズアニメーション（SVG / opacity: 0.02 / 8秒ループ）
- [ ] ゴールドの 0.5px ラインが縦に走るセクション区切り
- [ ] スクロール連動で画像が「浮き上がる」parallax（transformのみ）
- [ ] フッターに「夜の格言」的な1行テキストをイタリックで大きく配置
- [ ] カーソル近傍だけ画像の grayscale が解除されるホバー演出（JS可）
```
