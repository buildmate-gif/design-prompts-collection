# Pattern 05 — "Data Humanist"
**テーマ：データ×人間的温かみ / AI・テック・リサーチ発表に最適**

```
【UI/UXデザイン要件 — Data Humanist】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0. 出力形式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 出力：HTML + CSS（単一ファイル）または JSX + Tailwind CSS（単一ファイル）
- 外部フォント：Google Fonts（Noto Sans JP × Source Serif 4 × DM Mono）
- JavaScript：データビジュアライゼーション・カウントアップのみ許可

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 参照スタイル・世界観
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 参照：FT（Financial Times）/ The Economist / WIRED Japan
- キーワード：Humanized Data / Clarity with Warmth / Evidence-Based Narrative
- 一言で表すなら「数字が語る、でも冷たくない。データが人間の物語になる瞬間」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. カラーパレット（厳守）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- ベース：ウォームペーパー #F4F0E8 / ライトサンド #EAE5DB
- テキスト：ディープチャコール #1E1B14 / ウォームグレー #6E6860
- アクセント：1色のみ（FTサーモン #E87B52 またはFT青緑 #2E7D6B から選択）
  ※ グラフ・数値・強調キーワードにのみ使用
- 禁止：純白、純黒、コールドグレー（青みがかったグレー）、彩度の高い複数色使い

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. タイポグラフィ（階層定義）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 見出し（H1）：Source Serif 4 / font-size: clamp(2rem, 4.5vw, 3.5rem)
  font-weight: 300 / letter-spacing: -0.01em（新聞見出しの落ち着き）
- 数値・KPI：DM Mono / font-size: clamp(3rem, 8vw, 7rem) / font-weight: 400
  color: アクセントカラー / これがビジュアルの主役
- 見出し（H2）：Noto Sans JP / font-size: 0.8rem / letter-spacing: 0.15em / uppercase
  アクセントカラーの細いボーダー左付き（border-left: 2px solid）
- 本文（body）：Noto Sans JP / font-size: 1rem / line-height: 1.9
- DM Monoの大型数値と Source Serif の見出しで「記事×データ」の二重構造を作ること

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. レイアウト原則
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- グリッド：[2col 数値] + [10col テキスト] / [6col チャート] + [6col 解説] など変化させる
- コンテンツ幅：最大 1000px（読みやすさ優先）
- データセクション：数値を左に大きく配置し、右に解説テキストを添える構造を基本とする
- セクション間は 1px の水平ルーラー + セクション番号（01 / 02…）で区切ること
- Whitespace：余白は全体の30%以上を維持する（情報密度を意図的に抑える）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 装飾ルール（情報設計の美学）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【使用許可】
- アクセントカラーの border-left（2px）によるデータラベル表現
- カウントアップアニメーション（大型数値、スクロール時に発火）
- 薄いグリッド線（#1E1B1408）による背景のグラフ用紙的質感
- テキストのフェードイン（0.6s ease-out / スクロール連動）
- DM Monoの数値に対する下線強調（text-decoration / アクセントカラー）

【完全禁止】
- 派手なチャート・3Dグラフ（シンプルなSVGラインのみ許可）
- インフォグラフィック的な「賑やかな」レイアウト
- 複数のアクセントカラー同時使用
- アイコン・絵文字
- フラットデザイン的な色面の多用（情報の「重み」を失わせる）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. 「忘れられないディテール」の要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
以下のうち最低2つを必ず実装すること：
- [ ] DM Monoの大型数値（例：1,247社 / 92% / ¥48.2億）のカウントアップアニメーション
- [ ] SVGによる手書き的なラインチャート（シンプルな折れ線 / アクセントカラー1色）
- [ ] 印刷物的なフォリオ表現：右下固定の「P.01」スタイルページ番号
- [ ] 引用ブロック：border-left 3px + Source Serif italic で「証言」的な演出
- [ ] グリッド背景（background-image: repeating-linear-gradient で薄いガイドライン）
```
