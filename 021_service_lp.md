# シーン1：サービス紹介LP プロンプト（リデザイン版）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、サービス紹介ランディングページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- サービス名：[サービス名]
- キャッチコピー：[キャッチコピー]
- サービス概要：[2〜3文でサービスの説明]
- 特徴・強み：[特徴1] / [特徴2] / [特徴3]
- ターゲット：[想定ユーザー像]
- CTA文言：[ボタンテキスト]
- 会社名 / 提供者名：[名前]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Space Grotesk と Syne のみ使用）

【世界観・参照スタイル】
- ベルリンのテックスタートアップ / 北欧のSaaS企業サイトを参照
- 「機能美と余白が共存する、無駄のない緊張感」
- Dark Mode / ハイコントラスト / Brutalist Grid

【カラーパレット（厳守）】
- ベース：ディープチャコール #0F0F0F / ダークグレー #1C1C1C
- サーフェス：#242424 / #2E2E2E
- テキスト：オフホワイト #F0EDE8 / ミッドグレー #888884
- アクセント：1色のみ（電気グリーン #00FF87 または アシッドイエロー #E8F020 から選択）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- H1：Syne / clamp(3rem, 8vw, 7rem) / font-weight: 800 / letter-spacing: -0.04em / uppercase
- H2：Space Grotesk / clamp(1.4rem, 3vw, 2.2rem) / font-weight: 500
- 本文：Space Grotesk / 1rem / line-height: 1.7 / letter-spacing: 0.01em
- ラベル：Syne / uppercase / 0.65rem / letter-spacing: 0.2em
- アクセントカラーで特定の単語だけ着色し、テキスト内にコントラストを作ること

【レイアウト原則】
- CSS Grid（columns: repeat(24, 1fr)）の超密グリッド
- ヒーローエリア：テキストを左寄せ、右側2/3に大型のサービスビジュアルエリア
- セクション間の境界線：2px solid #F0EDE808（極細ライン）
- コンテンツは常に左揃え、中央揃え禁止
- 数字・実績エリアは横幅いっぱいのフルブリードで配置

【ページ構成（この順番で実装）】
1. Hero：H1（uppercase）+ キャッチコピー + CTAボタン（アクセントカラー）
2. Ticker：横スクロールするテキストマーキー（サービス名・キーワードの繰り返し）
3. 課題提起：3つの問いを番号付きで縦に並べる
4. サービス特徴：3項目を左右交互のジグザグレイアウトで
5. 実績数値：フルブリードの数字グリッド
6. CTA：ダークな背景にアクセントカラーのボタン1つ

【装飾・インタラクション】
- ヒーローのH1：文字が1文字ずつ上からフォールインするアニメーション
- マーキー（Ticker）：CSS animation による無限横スクロール
- CTAボタン：ホバー時に背景とテキストが反転するinvert効果
- カーソル：カスタムサークルカーソル（アクセントカラー、20px）

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超（ピル型・カード型禁止）
- 彩度の高いグラデーション（背景への使用禁止）
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui（汎用フォント禁止）
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
```
