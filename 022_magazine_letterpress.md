# シーン2：マガジン / 長文記事ページ プロンプト（リデザイン版）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、マガジン／長文記事ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 記事タイトル：[記事タイトル]
- サブタイトル：[サブタイトルまたはリード文]
- 著者名：[著者名]
- カテゴリ：[カテゴリ名]
- 公開日：[YYYY.MM.DD]
- 本文：[記事本文をここに貼り付け、または「ダミーテキストで作成」と指示]
- 中見出し（H2）：[見出し1] / [見出し2] / [見出し3]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Lora + Libre Baskerville + Nunito Sans）のみ使用

【世界観・参照スタイル】
- 1970年代のアメリカ文芸誌 / 独立系リトルプレスを参照
- 「インクの匂いがしそうな、アナログの温度感」
- Warm Analog / 活版印刷的テクスチャ / ローファイ・エディトリアル

【カラーパレット（厳守）】
- ベース：クリーム #FAF6EE / ウォームペーパー #F2EBD9
- テキスト：インクブラウン #1F1208 / ミッドブラウン #6B5744
- ボーダー・区切り：#C4A882
- アクセント：1色のみ（テラコッタ #C05A2A）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- 記事タイトル（H1）：Libre Baskerville / clamp(2.2rem, 5vw, 4rem) / font-style: italic / font-weight: 700
- 中見出し（H2）：Lora / clamp(1.3rem, 2.5vw, 1.9rem) / font-weight: 600
- 本文：Lora / 1.05rem / line-height: 1.95 / letter-spacing: 0.015em
- プルクオート：Libre Baskerville italic / 1.5rem / color: テラコッタ
- メタ情報（著者・日付）：Nunito Sans / uppercase / 0.72rem / letter-spacing: 0.14em

【レイアウト原則】
- 本文カラム：最大 660px / ページ中央
- 記事タイトルは本文幅より広い140%幅ではみ出させる
- 2段組レイアウトを中盤1セクションだけ使用（newspaper column）
- セクション間の仕切りは活版印刷的な飾り記号（CSS で生成：ornament）
- 欄外（margin）にアノテーション的な補足テキストを配置

【ページ構成（この順番で実装）】
1. ヘッダー：カテゴリ + 日付（新聞的な上段）
2. タイトルエリア：H1（はみ出しレイアウト）+ リード文
3. 著者byline：著者名 + 短いプロフィール1行
4. 本文：H2で区切った複数セクション（中盤に2段組を1セクション）
5. プルクオート：テラコッタで大きく引用
6. フッター：著者情報ボックス + タグ一覧

【装飾・インタラクション】
- 記事タイトル下に手書き風アンダーライン（SVG path / stroke-dasharray アニメーション）
- スクロール量に応じて上部に読了バーを表示（テラコッタ色 / 2px）
- 脚注リンクホバー時：ツールチップがfade-inで表示
- 欄外テキスト：スクロールに合わせてopacityが変化

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割りカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1200px〜）
- モバイルでは2段組を1段組に切り替え
- 欄外テキストはモバイルでは本文内インラインに変換
```
