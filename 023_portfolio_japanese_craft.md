# シーン3：ポートフォリオ / 作品集ページ プロンプト（リデザイン版）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、ポートフォリオ／作品集ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 氏名 / 屋号：[名前]
- 職種 / 肩書き：[職種]
- キャッチフレーズ：[一言で表すキャッチ]
- 略歴：[2〜3文のプロフィール]
- 専門領域：[領域1] / [領域2] / [領域3]
- 掲載作品数：[数]件（画像はプレースホルダーで可）
- 作品情報：[作品1タイトル：カテゴリ：年] / [作品2タイトル：カテゴリ：年] / ...
- 連絡先：[メール or URL]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Zen Kaku Gothic New + Josefin Sans）のみ使用

【世界観・参照スタイル】
- 京都の工芸作家のアトリエ / 21_21 DESIGN SIGHTの展示構成を参照
- 「削ぎ落とした先にある、日本的クラフトマンシップの静けさ」
- Japanese Minimalism / Ma（間）の美学 / モノクロームとテクスチャ

【カラーパレット（厳守）】
- ベース：生成り #F7F4EF / 薄墨 #EBEBEB
- テキスト：墨 #1A1614 / 鼠 #7A7673
- ボーダー：#CECAC4
- アクセント：1色のみ（弁柄 #8B2E16 または 鉄紺 #1D2B4F から選択）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- 名前（H1）：Josefin Sans / clamp(1rem, 2vw, 1.4rem) / font-weight: 300 / uppercase / letter-spacing: 0.25em（あえて小さく、静かに）
- 作品タイトル：Zen Kaku Gothic New / clamp(1.1rem, 2vw, 1.6rem) / font-weight: 400
- 本文：Zen Kaku Gothic New / 0.95rem / line-height: 2.0 / letter-spacing: 0.05em
- カテゴリ・年：Josefin Sans / uppercase / 0.65rem / letter-spacing: 0.18em

【レイアウト原則】
- 全体を「間（ま）」のある縦スクロール構成
- 作品一覧は1カラム縦型リスト（左に番号・右に作品情報のT字型レイアウト）
- 各作品行：左15%に番号、左15〜50%に作品タイトル、右50〜100%に画像エリア
- 画像エリアはホバーで拡大表示（transform: scale）
- セクション間の余白：最低 120px / 大きな「間」を意図的に作る

【ページ構成（この順番で実装）】
1. Header：名前（小さめ）+ 職種（uppercase） / 右端に現在時刻を表示
2. Introduction：キャッチフレーズを1行大きく + 略歴（短く）
3. Works：縦型T字レイアウトで全作品を順に並べる
4. Profile：専門領域を横並びのシンプルなタグで
5. Contact：メールアドレスのみ、大きく中央に

【装飾・インタラクション】
- ページロード時：名前とナビが上からゆっくりfade-in（1.2s ease）
- 作品行ホバー：画像がscale(1.03)、タイトルがアクセントカラーに変色
- スクロール：各作品行がviewportに入ると左からslide-in
- ヘッダー右端のリアルタイム時計：JavaScriptで動作

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは作品リストを画像上・テキスト下のスタック型に切り替え
```
