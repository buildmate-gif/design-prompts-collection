# シーン4：イベント / 告知ページ プロンプト（リデザイン版）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、イベント／告知ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- イベント名：[イベント名]
- キャッチコピー：[一言キャッチ]
- イベント概要：[2〜3文]
- 日時：[YYYY年MM月DD日（曜日） HH:MM〜HH:MM]
- 会場：[会場名 / 住所またはオンライン]
- 定員：[人数]名
- 参加費：[金額または無料]
- 対象者：[どんな人向けか]
- タイムテーブル：[時刻：内容] / [時刻：内容] / ...
- 申込みURL / 連絡先：[URL またはメール]
- 主催者名：[名前 / 団体名]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Anton + Barlow Condensed + Noto Sans JP）のみ使用

【世界観・参照スタイル】
- スイス国際タイポグラフィ / 1960年代のコンサートポスター / 現代フェスティビジュアルを参照
- 「壁に貼られたポスターが画面の中に来たような、物理的な迫力」
- Swiss Poster / Bold Typographic / 情報の優先順位を視覚的に徹底

【カラーパレット（厳守）】
- ベース：ポスターホワイト #F4F1EC
- テキスト：ポスターブラック #0D0D0D
- アクセントゾーン：イベントの雰囲気に合わせた1色（例：バーミリオン #CC3422 / ロイヤルイエロー #F0C040 から選択）
- アクセントゾーンの背景にはテキストを反転（ネガポジ）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- イベント名（H1）：Anton / clamp(4rem, 12vw, 10rem) / uppercase / letter-spacing: -0.02em（ポスター的圧縮）
- 日時（最重要情報）：Barlow Condensed / clamp(2rem, 5vw, 4rem) / font-weight: 700 / uppercase
- 本文・説明：Barlow Condensed / 1rem / line-height: 1.6 / font-weight: 400
- 日本語テキスト：Noto Sans JP / font-weight: 400
- ラベル（DATE / VENUE / FEE）：Anton / uppercase / 0.7rem / letter-spacing: 0.2em

【レイアウト原則】
- ファーストビューをフルスクリーンの「ポスター1枚」として設計
- イベント名（H1）を最上段、日時を2段目に大きく、CTA最下段
- タイムテーブル：左に時刻（Anton太字）/ 右に内容のシンプルな2カラム
- アクセントカラーのブロックを帯状に使い、ネガポジで情報を強調
- 余白ではなく「密度の緩急」でリズムを作る

【ページ構成（この順番で実装）】
1. Heroポスター：H1（巨大）+ 日時 + 会場 + CTAボタン（全て1画面）
2. 概要帯：アクセントカラー背景のフルブリード帯にキャッチコピー
3. タイムテーブル：2カラム縦型
4. 詳細情報：定員・参加費・対象者を横並びの数値グリッドで
5. 申込みCTA：大型ボタン + 主催者名

【装飾・インタラクション】
- ページロード時：H1が画面外から高速でスライドイン（0.4s cubic-bezier）
- CTAボタン：ホバーでネガポジ反転（背景↔テキスト）
- カウントダウンタイマー：イベント日まで DD:HH:MM:SS
- 概要帯：テキストが横マーキーで流れる

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割りカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルではH1のフォントサイズを clamp(3rem, 15vw, 5rem) に調整
```
