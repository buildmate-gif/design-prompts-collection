# シーン5：プロフィール / About ページ プロンプト（リデザイン版）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、プロフィール／Aboutページをhtml+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 氏名 / ブランド名：[名前]
- 肩書き：[肩書き]
- キャッチフレーズ：[自分を一言で表すフレーズ]
- ビジョン・理念：[大切にしていること 2〜3文]
- 略歴（箇条書き可）：[年：出来事] / [年：出来事] / ...
- 実績・数字：[数値1：説明] / [数値2：説明] / [数値3：説明]
- 専門領域：[スキル1] / [スキル2] / [スキル3]
- 連絡先：[URL またはメール]
- 写真：[画像URL または「プレースホルダー使用」]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（IBM Plex Mono + IBM Plex Sans JP）のみ使用

【世界観・参照スタイル】
- NASAの技術資料 / 研究者のアーカイブサイト / データジャーナリズムを参照
- 「人物をデータとして美しく可視化する、科学的なポートレート」
- Document Aesthetic / Monospace Grid / Information Architecture

【カラーパレット（厳守）】
- ベース：リサーチホワイト #F6F6F4 / ライトグレー #EEEEEC
- テキスト：データブラック #111110 / サブグレー #727270
- グリッド線：#DDDDD8
- アクセント：1色のみ（サイエンスブルー #1A5FA8 または フォレストグリーン #1A6640 から選択）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- 全テキストの基本：IBM Plex Mono（等幅）で統一し、ドキュメント感を演出
- 名前（H1）：IBM Plex Mono / clamp(2rem, 5vw, 4rem) / font-weight: 700
- 見出し（H2）：IBM Plex Mono / 1.1rem / uppercase / letter-spacing: 0.1em
- 本文・略歴：IBM Plex Mono / 0.95rem / line-height: 1.8
- 数値：IBM Plex Mono / clamp(2.5rem, 5vw, 4.5rem) / font-weight: 700 / color: アクセント
- 日本語テキスト：IBM Plex Sans JP / 0.95rem / line-height: 1.9

【レイアウト原則】
- 全体がドキュメント／スプレッドシートのような構造
- ページ上部に「ファイルヘッダー」（NAME / ID / DATE / VERSION などのメタ情報行）
- セクションは「section_01」「section_02」のように番号と区切り線で管理
- 実績数値：ボックスグリッド内に配置（罫線で区切られた表形式）
- 略歴：左に年、右に内容の2カラム年表（terminal log的なスタイル）

【ページ構成（この順番で実装）】
1. ファイルヘッダー：DOCUMENT / 名前 / 肩書き / 更新日 などメタ情報の帯
2. Overview：ビジョン・理念を「ABSTRACT」として1ブロック
3. 実績データ：罫線区切りの表形式グリッドで数値を整列
4. Timeline：年表形式の略歴（年 / 内容の2カラム）
5. Expertise：専門領域をタグ的に列挙（tag形式、角丸なし）
6. Contact：メタ情報スタイルで「CONTACT: [メール]」のみ

【装飾・インタラクション】
- ページロード時：テキストがタイプライター的に1行ずつ現れる（JS、各行0.05s遅延）
- 実績数値：カウントアップアニメーション（IntersectionObserver）
- 見出しにカーソルを当てると：アクセントカラーの下線がscaleX(1)に
- ページ全体に薄い1px グリッド線（CSS background-image: linear-gra...は使わず、CSSのborder-rightを活用した列区切りのみ）

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション（背景への使用禁止）
- スキルバー（progress bar による能力表示）
- 均等割りカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは年表の2カラムをスタック型に変換
- ファイルヘッダーはモバイルで縦並びに変換
```
