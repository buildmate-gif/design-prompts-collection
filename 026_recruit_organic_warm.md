# シーン6：採用 / 求人ページ プロンプト（v3）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、採用／求人ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 会社名 / チーム名：[名前]
- 採用キャッチコピー：[一言キャッチ]
- 会社・チームの紹介：[2〜3文]
- 働く環境・カルチャー：[特徴1] / [特徴2] / [特徴3]
- 募集職種：[職種1：業務内容] / [職種2：業務内容] / ...
- 求める人物像：[3〜5項目]
- 待遇・条件：[給与] / [勤務地] / [勤務形態] / [福利厚生など]
- 選考フロー：[ステップ1] → [ステップ2] → [ステップ3]
- 応募URL / 連絡先：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Fraunces + DM Sans + Noto Sans JP）のみ使用

【世界観・参照スタイル】
- Kinfolk誌 / Apartamento誌 / Scandinavian lifestyle brandを参照
- 「人と仕事の温度が伝わる、生活者目線のリクルートデザイン」
- Organic Warmth / Living Editorial / 自然素材のテクスチャ感

【カラーパレット（厳守）】
- ベース：ウォームリネン #F8F3EB / ライトサンド #EDE6D8
- テキスト：ディープブラウン #1E1208 / ウォームグレー #756B60
- ボーダー・仕切り：#C9BBA8
- アクセント：1色のみ（モスグリーン #4A6741）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- キャッチコピー（H1）：Fraunces / clamp(2.8rem, 6vw, 5.5rem) / font-style: italic / font-weight: 300
- セクション見出し（H2）：Fraunces / clamp(1.6rem, 3vw, 2.4rem) / font-weight: 400
- 本文：DM Sans + Noto Sans JP / 1rem / line-height: 1.9 / letter-spacing: 0.02em
- 職種名・ラベル：DM Sans / uppercase / 0.72rem / letter-spacing: 0.16em / color: モスグリーン
- 数字・実績：Fraunces / clamp(2.5rem, 5vw, 4rem) / font-weight: 200

【レイアウト原則】
- 12カラムCSS Gridベース
- ヒーローエリア：テキスト左寄せ、右60%に大きな余白（画像プレースホルダー）
- カルチャー紹介は横幅をはみ出す「フルブリード写真帯」を1箇所使用
- 募集職種：アコーディオン形式（クリックで詳細が展開）
- 選考フロー：横並びのステップ表示（番号 → テキストの連結）
- セクション間の余白：最低 10rem

【ページ構成（この順番で実装）】
1. Hero：キャッチコピー（H1）+ 会社紹介1文 + 「話を聞く」CTA
2. Culture：カルチャー3項目を非均等レイアウトで（フルブリード帯）
3. Positions：募集職種のアコーディオンリスト
4. Requirements：求める人物像をシンプルなチェックリスト的表現（bullet禁止・番号付き段落）
5. Conditions：待遇・条件を整理したグリッド
6. Flow：選考フローの横ステップ
7. CTA：応募ボタン + 「まず話を聞くだけでもOK」のサブコピー

【装飾・インタラクション】
- アコーディオン：クリックで max-height アニメーション展開（JS）
- カルチャー帯：スクロール連動でゆっくり横移動（parallax / transform のみ）
- CTAボタン：ホバーでモスグリーン塗りに変化（transition: 0.3s）
- 各セクション：IntersectionObserver で opacity + translateY のfade-in

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは選考フローを縦型ステップに変換
```
