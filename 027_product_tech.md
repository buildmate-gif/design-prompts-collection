# シーン7：商品 / プロダクト紹介ページ プロンプト（v3）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、商品／プロダクト紹介ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 商品名 / プロダクト名：[名前]
- キャッチコピー：[一言キャッチ]
- 商品概要：[2〜3文]
- 特徴・仕様：[特徴1：説明] / [特徴2：説明] / [特徴3：説明]
- ターゲットユーザー：[どんな人向けか]
- 価格：[価格または「お問い合わせ」]
- CTA文言：[購入・問い合わせなど]
- ブランド名：[名前]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Outfit + Noto Sans JP）のみ使用

【世界観・参照スタイル】
- Apple製品ページ / Dyson / Arc'teryx のプロダクトサイトを参照
- 「素材と機能が語る、説明不要の完成度」
- Material Confidence / Product Focus / ディテールへの偏愛

【カラーパレット（厳守）】
- ベース：アッシュホワイト #F2F2F0 / ライトスモーク #E6E6E4
- テキスト：スレート #161618 / ミッドスレート #72727A
- ボーダー：#D0D0CE
- アクセント：1色のみ（商品の世界観に合わせて自由選択、例：スティールブルー #1E4A6E）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- 商品名（H1）：Outfit / clamp(3rem, 7vw, 6rem) / font-weight: 200 / letter-spacing: -0.03em
- 特徴見出し（H2）：Outfit / clamp(1.8rem, 3.5vw, 3rem) / font-weight: 300
- 本文：Outfit + Noto Sans JP / 1rem / line-height: 1.75 / letter-spacing: 0.01em
- 仕様ラベル：Outfit / uppercase / 0.68rem / letter-spacing: 0.18em
- 価格：Outfit / clamp(2rem, 4vw, 3.5rem) / font-weight: 600

【レイアウト原則】
- ファーストビュー：商品名（H1）+ 大型画像プレースホルダーが画面全体を占有
- 特徴紹介：左右交互の「テキスト50% / ビジュアル50%」交互レイアウト（画像はスクロールで固定 position: sticky）
- 仕様：横スクロール可能な比較テーブル（罫線のみ、背景色なし）
- CTA：価格と購入ボタンを画面下部に固定（sticky footer / モバイル優先）

【ページ構成（この順番で実装）】
1. Hero：商品名（大型H1）+ キャッチコピー + 画像プレースホルダー（フルスクリーン）
2. Intro：商品概要を簡潔に（最大3文）
3. Features：特徴を左右交互レイアウトで（stickyビジュアル付き）
4. Specs：仕様をテーブル形式で整理
5. Target：ターゲットユーザーをシナリオ形式で描写
6. CTA：価格 + 購入ボタン + サブコピー

【装飾・インタラクション】
- Heroスクロール：商品名がゆっくりfade-outしながら商品画像が前に出てくる
- Features：画像エリアが position: sticky でテキストに対して固定
- 仕様テーブル：横スクロールバーをカスタムスタイル（アクセントカラー）
- CTAボタン：ホバーで背景がアクセントカラーに塗りつぶし（transition: 0.25s）
- Sticky購入バー：スクロール量が一定を超えたら下から出現

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは左右交互レイアウトをスタック型に変換
- Sticky購入バーはモバイルで常時表示
```
