# シーン8：会社 / チーム紹介ページ プロンプト（v3）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、会社／チーム紹介ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 会社名 / チーム名：[名前]
- ミッション・理念：[一言または1文]
- 会社概要：[2〜3文]
- 創業年 / 設立背景：[年：経緯]
- バリュー（大切にしていること）：[バリュー1：説明] / [バリュー2：説明] / [バリュー3：説明]
- チームメンバー：[名前：役職：一言コメント] / ...（ダミー可）
- 実績・数字：[数値1：説明] / [数値2：説明] / [数値3：説明]
- 沿革：[年：出来事] / [年：出来事] / ...
- 連絡先 / URL：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Unbounded + Noto Serif JP）のみ使用

【世界観・参照スタイル】
- 建築事務所のアニュアルレポート / 現代美術館のコーポレートサイトを参照
- 「コンクリートと光のように、強さと透明感が共存する」
- Raw Architecture / Structural Honesty / 構造をそのままデザインにする

【カラーパレット（厳守）】
- ベース：コンクリートグレー #EBEBEA / ライトコンクリート #F2F2F1
- テキスト：ストラクチャルブラック #0A0A0A / ミッドグレー #707070
- アクセントライン：#B0B0AE
- アクセント：1色のみ（ブリックレッド #9E2A1C または インダストリアルゴールド #B08A3E から選択）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- 会社名・ミッション（H1）：Unbounded / clamp(2rem, 5vw, 4.5rem) / font-weight: 900 / uppercase / letter-spacing: -0.02em
- セクション見出し（H2）：Unbounded / clamp(1rem, 2vw, 1.6rem) / font-weight: 400 / uppercase
- 本文：Noto Serif JP / 1rem / line-height: 1.95 / letter-spacing: 0.04em
- バリュー番号：Unbounded / clamp(4rem, 8vw, 7rem) / font-weight: 900 / opacity: 0.07（背景的に大きく）
- ラベル：Unbounded / uppercase / 0.62rem / letter-spacing: 0.22em

【レイアウト原則】
- 全体をグリッドの「構造線」が支配するレイアウト
- ヒーロー：ミッション文を画面いっぱいに、余白は最小限
- バリューセクション：大きな数字（01/02/03）を背景的に配置し、テキストをオーバーレイ
- メンバー紹介：縦型リスト（名前・役職・コメントの3行セット、写真プレースホルダー左）
- 沿革：左に年・右に出来事、縦ライン1本で繋ぐタイムライン

【ページ構成（この順番で実装）】
1. Hero：ミッション（大型H1）+ 会社名・設立年
2. Overview：会社概要（簡潔に）+ 実績数値グリッド
3. Values：3つのバリューを大型番号付きで
4. Members：チームメンバーリスト
5. History：縦型タイムライン（沿革）
6. Contact：ミッションを再掲 + CTA

【装飾・インタラクション】
- ヒーロー：ページロード時にミッション文が左から1単語ずつ現れる
- バリュー番号：スクロール連動でゆっくり回転（transform: rotate / 最大15deg）
- メンバー写真：ホバーでグレースケールが解除（filter: grayscale）
- タイムライン縦ライン：スクロールに合わせて上から伸びるアニメーション

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルではメンバーリストを画像上・テキスト下のスタック型に
```
