# シーン9：価格 / 料金プランページ プロンプト（v3）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、価格／料金プランページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- サービス名：[名前]
- 料金ページのキャッチ：[一言キャッチ]
- プラン1：[プラン名：価格：対象：含まれる機能リスト]
- プラン2：[プラン名：価格：対象：含まれる機能リスト]（推奨プラン）
- プラン3：[プラン名：価格：対象：含まれる機能リスト]
- FAQ：[質問1：回答] / [質問2：回答] / [質問3：回答]
- 保証・サポート：[返金保証・サポート体制など]
- CTA文言：[ボタンテキスト]
- 連絡先：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Epilogue + Noto Sans JP）のみ使用

【世界観・参照スタイル】
- The Economist / Financial Times のデジタル版 / 高級プライベートバンクのサービス案内を参照
- 「数字と誠実さで信頼を積む、迷いのない情報設計」
- Financial Clarity / Trust Architecture / 情報の優先順位が完璧に整理された美しさ

【カラーパレット（厳守）】
- ベース：ニュートラルホワイト #F5F5F3 / ライトグレー #EDEDEB
- テキスト：チャコール #141414 / ミッドグレー #686866
- ボーダー・罫線：#D4D4D2
- 推奨プランのハイライト：背景 #1A1A1A / テキスト #F5F5F3（ダーク反転）
- アクセント：1色のみ（アンバーゴールド #C4870A）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- ページキャッチ（H1）：Epilogue / clamp(2.2rem, 5vw, 4rem) / font-weight: 800 / letter-spacing: -0.02em
- プラン名（H2）：Epilogue / clamp(1.1rem, 2vw, 1.6rem) / font-weight: 700 / uppercase
- 価格数値：Epilogue / clamp(2.5rem, 5vw, 4rem) / font-weight: 800
- 本文・機能リスト：Epilogue + Noto Sans JP / 0.95rem / line-height: 1.75
- FAQ見出し：Epilogue / 1.05rem / font-weight: 600
- ラベル：Epilogue / uppercase / 0.65rem / letter-spacing: 0.16em / color: アンバーゴールド

【レイアウト原則】
- プランは3カラムだが**均等割り禁止**：幅比を [1fr : 1.1fr : 1fr] で推奨プランを中央強調
- 推奨プランのみ背景をダーク反転（#1A1A1A）して視覚的に浮き上がらせる
- 機能比較は横断的な比較テーブルをプランカードの下に別途配置
- FAQ：アコーディオン形式（1問ずつ展開）
- 月額 / 年額の切り替えトグル（JS）を価格の上部に配置

【ページ構成（この順番で実装）】
1. キャッチ：H1 + サブコピー（1〜2文）
2. 月額/年額トグル：切り替えで価格が変動
3. プランカード：3プランを横並びで（推奨プランをダーク反転）
4. 比較テーブル：全機能を横断比較（チェックマークの代替として「—」と「●」のテキスト記号のみ使用）
5. FAQ：アコーディオン
6. 保証・サポート：1行の安心訴求
7. CTA：最終コンバージョン

【装飾・インタラクション】
- 月額/年額トグル：切り替え時に価格が数字を上下にスライドしながら変化
- FAQアコーディオン：max-height アニメーション展開（0.35s ease）
- プランカードホバー：推奨プラン以外はborderがアンバーゴールドに変化
- 比較テーブル：スクロール時に列ヘッダーがsticky

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- チェックマーク記号（✓ ✗ など）の使用禁止（テキスト「●」「—」で代替）
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード（幅差をつけること）
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルではプランカードを縦積みに変換
- 比較テーブルは横スクロール対応
```
