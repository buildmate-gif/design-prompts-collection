X-height

エックスハイト / 小文字の高さ

欧文フォントにおいて、ベースラインから小文字の「x」の上端までの高さを指す。アセンダーやディセンダーを持たない小文字（a, c, e, m, n, oなど）の基準となる高さ。x-heightが大きいほど小文字が大きく見え、小さなサイズでも可読性が高まる傾向がある。

See also: Baseline, Cap Height
Spacing

Kerning

カーニング / 文字詰め

特定の文字の組み合わせ（ペア）において、文字間の空きを視覚的に均一に見えるように調整する作業。例えば「A」と「V」や「T」と「o」など、物理的な文字幅のまま配置すると隙間が空きすぎて見えるペアに対して、文字間を詰める物理的・デジタル的処理。

See also: Tracking, Ligature
Spacing

Leading

レディング / 行送り

行と行の間隔のこと。もともとは活版印刷において、行間に挿入された鉛（Lead）の薄い板に由来する。ベースラインから次の行のベースラインまでの距離を表し、CSSの `line-height` の概念に近い。長文の可読性に重大な影響を与える。

See also: Baseline
Classification

Serif

セリフ / 飾り線

文字のストロークの端にある小さな飾り線や意匠。「サンセリフ（Sans-serif）」は逆にこの飾りがない書体を指す。ローマ時代の石版彫刻の筆跡に由来するとされ、長文の印刷物において視線を誘導し可読性を高める効果があると言われている。

See also: Sans-serif
Letter Anatomy

Counter

カウンター / ふところ

「o」や「e」、「b」などの文字の内部にある、完全または部分的に閉じられた「空白」の部分。日本語フォントにおける「ふところ」に近い概念。カウンターが広く開いている書体は、小さいサイズで印刷・表示された際にも潰れにくく視認性が高い。

See also: Aperture
