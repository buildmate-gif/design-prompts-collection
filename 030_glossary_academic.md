# シーン10：ナレッジ / 用語集ページ プロンプト（v3）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、ナレッジ／用語集ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- サイト名 / ページタイトル：[タイトル]
- テーマ・分野：[専門分野]（例：建設業界用語 / AIマーケティング用語）
- サブタイトル：[1文のリード]
- 用語データ（最低5件）：[用語名：読み方：説明：関連用語] / ...
- カテゴリ：[カテゴリ1] / [カテゴリ2] / [カテゴリ3]（用語をグルーピング）
- 運営者名：[名前 / 組織名]
- 連絡先 / URL：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Spectral + Source Sans 3 + Noto Serif JP）のみ使用

【世界観・参照スタイル】
- オックスフォード英英辞典のオンライン版 / 大学図書館の特別コレクションを参照
- 「知識の重みと、調べる喜びが共存するアーカイブ空間」
- Academic Archive / Reference Library / 学術的な静けさと奥行き

【カラーパレット（厳守）】
- ベース：パーチメント #F6F1E7 / ライトバフ #EDE7D6
- テキスト：インク #18120A / セピアグレー #6E6055
- ボーダー・仕切り：#C8BAA2
- アクセント：1色のみ（ディープトール #2A5C5A）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- ページタイトル（H1）：Spectral / clamp(2rem, 4.5vw, 3.8rem) / font-style: italic / font-weight: 700
- 用語名（H2）：Spectral / clamp(1.4rem, 2.5vw, 2rem) / font-weight: 600
- 読み方（ふりがな的）：Source Sans 3 / 0.75rem / letter-spacing: 0.1em / color: セピアグレー
- 説明文：Source Sans 3 + Noto Serif JP / 1rem / line-height: 1.95 / letter-spacing: 0.02em
- カテゴリラベル：Source Sans 3 / uppercase / 0.65rem / letter-spacing: 0.16em
- 関連用語リンク：Spectral italic / 0.9rem / color: ディープトール / underline

【レイアウト原則】
- 2ペインレイアウト：左サイドバー（索引ナビ）+ 右メインコンテンツ（7:3 split）
- 左サイドバー：アルファベット / 五十音 索引 + カテゴリフィルター（sticky）
- 用語エントリー：辞書的な縦積みリスト（用語名 → 読み → 説明 → 関連用語）
- 各エントリーは上部に1pxの仕切り線
- キーワード検索フィールドをサイドバー上部に配置（JS でリアルタイムフィルタ）

【ページ構成（この順番で実装）】
1. ヘッダー：タイトル（H1）+ テーマ・分野 + 用語件数表示
2. 2ペインレイアウト開始
   - 左：検索フィールド + カテゴリフィルター + 索引ナビ（sticky）
   - 右：用語エントリーの縦リスト
3. 各エントリー：用語名 / 読み / 説明本文 / 関連用語
4. フッター：運営者名 + 最終更新日

【装飾・インタラクション】
- 検索フィールド：入力するたびにエントリーがリアルタイムでフィルタリング（JS）
- カテゴリフィルター：クリックで該当エントリーのみ表示
- 索引ナビ（あ行/か行… or A/B/C…）：クリックで該当エントリーへスムーズスクロール
- エントリーホバー：背景がライトバフに変化、左border 2px ディープトールが出現
- 関連用語クリック：該当エントリーへスムーズスクロール + ハイライト（0.5s後に解除）

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割りカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルではサイドバーを上部の横スクロールタブ（カテゴリ）＋ドロワー式索引に変換
- 2ペインを1カラムに切り替え
```
