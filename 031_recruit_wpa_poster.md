# シーン6：採用 / 求人ページ プロンプト（v4）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、採用／求人ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 会社名 / チーム名：[名前]
- 採用キャッチコピー：[一言キャッチ]
- 会社・チームの紹介：[2〜3文]
- 働く環境・カルチャー：[特徴1] / [特徴2] / [特徴3]
- 募集職種：[職種1：業務内容] / [職種2：業務内容] / ...
- 求める人物像：[3〜5項目]
- 待遇・条件：[給与] / [勤務地] / [勤務形態] / [福利厚生]
- 選考フロー：[ステップ1] → [ステップ2] → [ステップ3]
- 応募URL / 連絡先：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Big Shoulders Display + Noto Sans JP）のみ使用

【世界観・参照スタイル】
- 1920年代のアメリカ鉄道ポスター / WPA（公共事業促進局）アート / ニューディール期のプロパガンダデザインを参照
- 「働くことへの誇りを、力強いビジュアルで呼び起こす」
- Bold Heroism / Geometric Deco / 労働の美学

【カラーパレット（厳守）】
- ベース：ディープネイビー #0E1C2E / ミッドネイビー #162540
- サーフェス：#1E3050
- テキスト：クリームホワイト #F0EBE0 / ミッドクリーム #B8AE9E
- アクセント：1色のみ（ゴールデンアンバー #D4900A）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- キャッチコピー（H1）：Big Shoulders Display / clamp(4rem, 10vw, 9rem) / font-weight: 900 / uppercase / letter-spacing: -0.01em
- セクション見出し（H2）：Big Shoulders Display / clamp(2rem, 4vw, 3.5rem) / font-weight: 700 / uppercase
- 本文：Noto Sans JP / 1rem / line-height: 1.85 / letter-spacing: 0.03em
- 職種名：Big Shoulders Display / 1.4rem / uppercase / color: ゴールデンアンバー
- ラベル：Big Shoulders Display / uppercase / 0.7rem / letter-spacing: 0.2em

【レイアウト原則】
- 全体をダークモードで構築（ベースはネイビー系）
- ヒーロー：H1を画面中央に巨大配置、上下に細い水平線（1px ゴールデンアンバー）
- カルチャーセクション：縦型タイル（各タイルは大きな数字 + 短いテキスト）
- 募集職種：ゴールデンアンバーのラインで区切ったリスト形式
- 選考フロー：横並び、番号を大きく、矢印は「→」テキストのみ

【ページ構成（この順番で実装）】
1. Hero：H1（巨大）+ 会社名 + 水平線 + CTA
2. Mission帯：フルブリードのゴールデンアンバー帯にミッション文（ダーク反転）
3. Culture：数字タイル3枚（縦型）
4. Positions：職種リスト（展開式アコーディオン）
5. Conditions：待遇・条件をグリッドで
6. Flow：選考ステップ横並び
7. CTA：H1を小さく再掲 + 応募ボタン

【装飰・インタラクション】
- ページロード：H1が下から高速スライドイン（0.5s cubic-bezier(0.16,1,0.3,1)）
- Mission帯：スクロールでテキストが左から右へ流れるマーキー
- 職種アコーディオン：クリックでゴールデンアンバーの左ボーダーが出現しながら展開
- CTAボタン：ホバーでゴールデンアンバー塗り + テキストがネイビーに反転

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは選考フローを縦型ステップに変換
```
