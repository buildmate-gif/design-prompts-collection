# シーン8：会社 / チーム紹介ページ プロンプト（v4）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、会社／チーム紹介ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- 会社名 / チーム名：[名前]
- ミッション・理念：[一言または1文]
- 会社概要：[2〜3文]
- 創業年 / 設立背景：[年：経緯]
- バリュー：[バリュー1：説明] / [バリュー2：説明] / [バリュー3：説明]
- チームメンバー：[名前：役職：一言コメント] / ...（ダミー可）
- 実績・数字：[数値1：説明] / [数値2：説明] / [数値3：説明]
- 沿革：[年：出来事] / [年：出来事] / ...
- 連絡先 / URL：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（VT323 + Space Mono + Noto Sans JP）のみ使用

【世界観・参照スタイル】
- 1980年代のテクノロジー雑誌 / BASICプログラムの印刷物 / 初期インターネットのZINE文化を参照
- 「ドット・グリッド・端末画面の懐かしさと、今の誠実さが混ざり合う」
- Retro Terminal / Tech Zine / ピクセルとミニマルの共存

【カラーパレット（厳守）】
- ベース：グリッドペーパー #F0F0EE / ライトグリッド #E8E8E6
- テキスト：ターミナルブラック #111111 / フェードグレー #626260
- グリッド線：#D0D0CE（1px、background-imageではなくborderで実装）
- アクセント：1色のみ（フォスフォグリーン #2A7A3A）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- ミッション・会社名（H1）：VT323 / clamp(2.5rem, 6vw, 5rem) / letter-spacing: 0.04em（ドット感を活かす）
- セクション見出し（H2）：Space Mono / clamp(1rem, 2vw, 1.5rem) / font-weight: 700 / uppercase
- 本文：Space Mono + Noto Sans JP / 0.9rem / line-height: 1.9 / letter-spacing: 0.02em
- 実績数値：VT323 / clamp(3rem, 6vw, 5rem) / color: フォスフォグリーン
- ラベル・コメント：Space Mono / uppercase / 0.65rem / letter-spacing: 0.18em
- 年号（沿革）：VT323 / 1.8rem / color: フォスフォグリーン

【レイアウト原則】
- ページ全体に1px グリッド線を背景として引く（borderのみ使用、背景画像禁止）
- ヒーロー：ミッション文を「> [ミッション文]_」のようにターミナルプロンプト形式で表示
- 実績数値：グリッドセルの中に数値を大きく配置（罫線で囲む）
- 沿革：左列に年号（VT323）、右列に出来事（Space Mono）の2カラム表
- メンバー紹介：名前・役職・コメントをコードコメント形式（/* name: / role: / note: */）で表現

【ページ構成（この順番で実装）】
1. Boot画面風ヘッダー：会社名 + 設立年 + バージョン番号（例：v2024.01）
2. Mission：ターミナルプロンプト形式のミッション表示
3. Stats：実績数値グリッド（罫線囲み）
4. Values：バリューをコードブロック風に（背景：#E8E8E6、左ボーダー：フォスフォグリーン）
5. Members：コードコメント形式のメンバー紹介リスト
6. History：年表テーブル
7. Contact：「CONTACT: [メール]」1行のみ

【装飾・インタラクション】
- ページロード：ミッション文がタイプライターアニメーションで1文字ずつ表示（VT323で）
- カーソル点滅：ミッション末尾に「_」がblinkするCSS animation
- メンバーホバー：コードコメントがフォスフォグリーンに変色
- スクロール：各セクション見出しの前に「> SECTION_XX_LOADING...」が一瞬表示されてからフェード

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは沿革テーブルを縦積みに変換
- グリッド線はモバイルで列数を減らす
```
