# シーン9：価格 / 料金プランページ プロンプト（v4）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、価格／料金プランページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- サービス名：[名前]
- 料金ページのキャッチ：[一言キャッチ]
- プラン1：[プラン名：価格：対象：含まれる機能リスト]
- プラン2：[プラン名：価格：対象：含まれる機能リスト]（推奨プラン）
- プラン3：[プラン名：価格：対象：含まれる機能リスト]
- FAQ：[質問1：回答] / [質問2：回答] / [質問3：回答]
- 保証・サポート：[返金保証・サポート体制など]
- CTA文言：[ボタンテキスト]
- 連絡先：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Didact Gothic + Cinzel + Noto Serif JP）のみ使用

【世界観・参照スタイル】
- パリのオートクチュールブランド / ホテルリッツのルームカテゴリページ / 高級腕時計ブランドのコレクション一覧を参照
- 「価格を恥じず、価値を誇る。選ぶこと自体が体験になるページ」
- Haute Couture / Vertical Luxury / 値段より格を伝える

【カラーパレット（厳守）】
- ベース：シャンパン #F7F3EC / パールグレー #EDE8DF
- テキスト：エボニー #111008 / ウォームグレー #706A60
- ボーダー：#C4B89A
- 推奨プランのハイライト：背景 シャンパンゴールド #C8A84B / テキスト #111008
- アクセント：1色のみ（シャンパンゴールド #C8A84B）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- ページキャッチ（H1）：Cinzel / clamp(2rem, 4.5vw, 4rem) / font-weight: 400 / letter-spacing: 0.06em（ローマン体の威厳）
- プラン名（H2）：Cinzel / clamp(1.1rem, 2vw, 1.6rem) / font-weight: 400 / letter-spacing: 0.1em / uppercase
- 価格数値：Didact Gothic / clamp(2.8rem, 5vw, 4.5rem) / font-weight: 400
- 本文・機能リスト：Didact Gothic + Noto Serif JP / 0.95rem / line-height: 1.85
- FAQ見出し：Cinzel / 1rem / letter-spacing: 0.06em
- ラベル（PLAN / PRICE / INCLUDES）：Cinzel / uppercase / 0.65rem / letter-spacing: 0.2em

【レイアウト原則】
- プランを横並びにせず「縦型カード積み」で提示（1プラン1行、幅いっぱい）
- 各プランはフルブリードの横帯として、内部を左右2分割（左：プラン名・価格 / 右：機能一覧）
- 推奨プランの帯のみシャンパンゴールド背景
- プラン間の区切りは 1px ボーダー + 大きな余白（min 5rem）
- FAQは縦型アコーディオン（Cinzel見出し）

【ページ構成（この順番で実装）】
1. キャッチ：H1 + 短いリード文（Cinzelで格調高く）
2. プラン一覧：縦積みフルブリード帯（3プラン）
3. 比較概要：プランの違いを3列テーブルで簡潔に（罫線のみ）
4. FAQ：Cinzel見出しのアコーディオン
5. 保証・サポート：1行の格調ある安心訴求
6. CTA：最終コンバージョン（シャンパンゴールドのボタン）

【装飾・インタラクション】
- ページロード：H1が opacity: 0 → 1 にゆっくりフェードイン（1.5s ease）
- プラン帯ホバー：推奨プラン以外はボーダーがシャンパンゴールドに変化（transition: 0.4s）
- FAQアコーディオン：max-height 展開（0.45s ease）
- CTAボタン：ホバーで背景が濃いゴールドに深化（#A88830 へ transition: 0.3s）

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- チェックマーク記号（✓ ✗ など）の使用禁止（テキスト「●」「—」で代替）
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割り3カラムカード（縦積みレイアウトを厳守）
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルではプラン帯の左右2分割を縦積みに変換
- 比較テーブルは横スクロール対応
```
