# シーン10：ナレッジ / 用語集ページ プロンプト（v4）

---

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、ナレッジ／用語集ページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- サイト名 / ページタイトル：[タイトル]
- テーマ・分野：[専門分野]
- サブタイトル：[1文のリード]
- 用語データ（最低5件）：[用語名：読み方：説明：関連用語] / ...
- カテゴリ：[カテゴリ1] / [カテゴリ2] / [カテゴリ3]
- 運営者名：[名前 / 組織名]
- 連絡先 / URL：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Playfair Display + Karla + Noto Serif JP）のみ使用

【世界観・参照スタイル】
- 探検家のフィールドノート / ナショナルジオグラフィックのデータビジュアル / 博物館の解説パネルを参照
- 「知識を探検するように読み進める、冒険の地図としてのリファレンスページ」
- Field Notes / Expedition Log / 発見の喜びを設計する

【カラーパレット（厳守）】
- ベース：フィールドカーキ #F4EFE4 / ライトカーキ #EBE4D4
- テキスト：インクダーク #1A140A / タン #7A6E5E
- ボーダー・仕切り：#BEB09A
- アクセント：1色のみ（エクスプローラーレッド #A02820）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- ページタイトル（H1）：Playfair Display / clamp(2.2rem, 5vw, 4rem) / font-weight: 700 / font-style: italic
- 用語名（H2）：Playfair Display / clamp(1.4rem, 2.5vw, 2rem) / font-weight: 700
- 読み方：Karla / 0.78rem / letter-spacing: 0.12em / color: タン / uppercase
- 説明文：Karla + Noto Serif JP / 1rem / line-height: 1.95 / letter-spacing: 0.02em
- カテゴリラベル：Karla / uppercase / 0.65rem / letter-spacing: 0.16em / color: エクスプローラーレッド
- 関連用語：Playfair Display italic / 0.9rem / color: エクスプローラーレッド / underline: 点線

【レイアウト原則】
- 全体をフィールドノートのページに見立てた「罫線付き」レイアウト
- 横線（1px #BEB09A）を等間隔に引いた背景（border-bottomのみで実装、background-image禁止）
- 左余白に用語番号（01 / 02...）を縦に並べる「マージン番号」スタイル
- 検索フィールドをページ最上部に配置（罫線内に埋め込まれた手書き入力欄風）
- カテゴリはタブ型ではなく、横並びテキストリンク（アンダーラインなし、ホバーでエクスプローラーレッド）

【ページ構成（この順番で実装）】
1. ヘッダー：H1（italic）+ テーマ・分野 + 用語件数（「xx entries」）
2. 検索 + カテゴリフィルター：罫線上部に配置
3. 用語エントリー：罫線背景上に縦積み
   - 左マージン：番号（01/02...）
   - 用語名（H2）+ 読み方
   - カテゴリラベル
   - 説明本文
   - 関連用語（点線下線リンク）
4. フッター：運営者名 + 最終更新日

【装飾・インタラクション】
- 検索：入力するたびにエントリーがリアルタイムフィルタリング（JS）
- カテゴリフィルター：クリックで該当エントリーのみ表示、番号は再採番
- エントリーホバー：左マージン番号がエクスプローラーレッドに変化
- 関連用語クリック：該当エントリーへスムーズスクロール + 左ボーダー2px がフラッシュ（0.8s後に消える）
- ページロード：エントリーが上から順に 0.05s 遅延で fade-in

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 16px 超
- 彩度の高いグラデーション
- 均等割りカード
- Inter / Roboto / Arial / system-ui
- 紫・青のグラデーション

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは左マージン番号を用語名の上部に移動
- カテゴリフィルターを横スクロールタブに変換
```
