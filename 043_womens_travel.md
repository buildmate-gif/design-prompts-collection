# F13 — Women's Travel LP
## SOLA TRAVEL / 女性向けひとり旅・少人数旅行プランニング

---

## 目的
女性のひとり旅・女性同士の少人数旅を専門にするトラベルプランニングサービスのLP。
「ひとりだから行ける」「女性だから気づける」旅の豊かさを伝えるページ。
安心・安全の信頼感と、「旅に出たくなる」憧れの世界観を両立させる。

---

## フォント
- 大見出し：Libre Baskerville Italic / size clamp(2.5rem,5.5vw,5rem) / letter-spacing 0.04em
- 地名・ルビ：Jost / weight 300 / size 0.72rem / letter-spacing 0.22em / uppercase
- 本文：Noto Sans JP / weight 300 / size 0.9rem / line-height 2.2 / letter-spacing 0.07em
- 数字・価格：DM Mono / size 0.78rem
- **禁止：Inter / Roboto / Arial / system-ui**

---

## カラー
```
--bg:     #F6F2EC  サンドペーパー
--bg2:    #EDE8DF  ウォームアイボリー
--bg-sky: #E8EEF4  ペールスカイ（空を連想）
--ink:    #262218  ダークブラウン
--mid:    #847868  ウォームグレー
--dusk:   #9878A0  トワイライトモーブ（夕暮れ）
--sand:   #C4A870  デザートサンド
```
純白・純黒・蛍光色 禁止。「旅先の夕暮れ」のような深みと柔らかさ。

---

## レイアウト
- 12カラムグリッド（均等割り禁止）
- section padding 9rem〜11rem（mobile 4rem）
- 地図的な視点：デスティネーションを「点」として非線形に配置する箇所を1つ
- 横長（16:9）のシネマティックな写真プレースホルダー + 縦長（2:3）の「旅人視点」写真の混在
- ペールスカイ背景セクションを1箇所（空・解放感の演出）

---

## セクション構成
1. **DEPARTURE** — 大きなLibre Italic見出し + 「今夜、どこに行きたいですか」+ 横長シネマ画像
2. **DESTINATIONS** — 旅先紹介6点（非均等グリッド / 地名upper / 「こんな旅」短文）
3. **WHY SOLA** — 女性専門である3つの理由（テキスト主体・セリフ体・余白）
4. **JOURNEY** — 旅の流れ（相談→プランニング→出発→帰国後）タイムライン形式
5. **VOICE** — 旅した女性の声（3〜4件 / 引用スタイル / 旅先名付き）
6. **PLAN CTA** — ペールスカイ背景 / 「あなたの旅を一緒に考えます」+ 相談フォームへ

---

## ボタン・インタラクション（女性向け）
- CTAボタン：dusk カラー背景 / 角丸なし / hover で background が sand に変わる（0.5s）
- 旅先カード：hover で画像が scale(1.04) + overlay にコピーがふわり現れる（opacity 0→1 / 0.5s）
- タイムライン：スクロールで各ステップが左から順にフェードイン（stagger 0.2s）
- 引用：スクロール時に opacity 0 + translateY(10px) → 1 + 0 / 1s ease でゆっくり着地

---

## 必須ディテール（最低2つ）
- [ ] シネマティック比率：ヒーロー画像を aspect-ratio: 21/9 でワイドに / 視差 0.25倍速
- [ ] 夕暮れグラデ背景：ヒーロー下部に linear-gradient（sky → bg）でグラデーションフェード
- [ ] 地名装飾：デスティネーション名を letter-spacing 0.3em / 極細 / 上部に小さく配置
- [ ] タイムラインアニメ：各ステップが順番に左から translateX(-15px)→0 で現れる

---

## 装飾ルール
**OK：** 極細ボーダー / 視差 / overlay / タイムライン / SVG noise opacity 0.03
**禁止：** 絵文字 / アイコン / box-shadow blur 8px超 / 均等割り / 過剰なカラフル装飾

---

## 感情設計（コピー指針）
- 安全への不安を先回りして消す：「女性スタッフが同行」「24時間サポート」は事実として
- 「自分のためだけの旅」という自己肯定の文脈
- 旅先は「観光地」より「感情」で語る：「プラハの石畳で、なぜか泣いた」
- 「ひとりは孤独」という先入観を静かに壊す

---

## 補足
- ブランド名：**SOLA TRAVEL** / サービス名：架空（例：SOLA Select / SOLA Solo Plan）
- 旅先：架空でOK（Lisbon / Tbilisi / Luang Prabang / Oaxaca等のニッチな都市）
- コピー：日本語主体（地名・見出しは英語）
- 画像：CSSプレースホルダー（空・砂漠・海・夕暮れを連想するグラデーション）
