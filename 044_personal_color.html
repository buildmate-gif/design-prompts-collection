Four Seasons

Type 01

Spring

春の光のような、明るくクリアな色合い。イエローベースで彩度が高く、フレッシュで若々しい印象を与えます。

Type 02

Summer

初夏の紫陽花のような、パステルやスモーキーな寒色系。ブルーベースの色が、肌の透明感を美しく引き立てます。

Type 03

Autumn

紅葉や実りの秋を思わせる、深みのあるリッチなアースカラー。落ち着きとおだやかな温かさを醸し出します。

Type 04

Winter

雪景色に映えるような、ドラマティックで鮮やかな色。はっきりとしたコントラストが、シャープな洗練を与えます。
