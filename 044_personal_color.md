# F14 — Personal Color Diagnosis Service LP
## CHROMĒ / パーソナルカラー診断・スタイリングサービス

---

## 目的
パーソナルカラー診断＋スタイリングアドバイスを提供するサービスのLP。
「似合う色がわかると、鏡を見るのが楽しくなる」という体験価値を伝えるページ。
科学的根拠と美的感覚を両立させ、「なんとなく」ではなく「確信」を持って服を選べるようになる。

---

## フォント
- 大見出し：Cormorant Garamond / weight 300 / size clamp(2.5rem,6vw,5.5rem) / letter-spacing 0.05em
  - ただしイタリックは使わない（Upright体で品格・知性を表現）
- 中見出し：Noto Sans JP / weight 400 / letter-spacing 0.15em
- 本文：DM Sans / weight 300 / size 0.88rem / line-height 2.2 / letter-spacing 0.06em
- 色名・カテゴリ：Jost / size 0.6rem / letter-spacing 0.25em / uppercase
- **禁止：Inter / Roboto / Arial / system-ui**

---

## カラー
```
--bg:       #FAF8F5  ピュアホワイト寄りクリーム
--bg2:      #F2EDE8  ウォームクリーム
--ink:      #221C1A  ディープブラウン
--mid:      #907878  ウォームグレーローズ
--spring:   #E8B4B0  ソフトコーラル（春タイプ）
--summer:   #B4C4D8  アッシュラベンダー（夏タイプ）
--autumn:   #C89060  バーントオレンジ（秋タイプ）
--winter:   #4A4460  ディープパープル（冬タイプ）
```
4シーズンカラーをアクセントとして活用。メインはニュートラルクリーム系で統一。

---

## レイアウト
- 12カラムグリッド（均等割り禁止）
- section padding 9rem〜11rem（mobile 4rem）
- 4シーズンセクション：それぞれのパレットカラーを背景にした縦長カードを非均等配置
- カラースウォッチ（色見本）をUI要素として積極活用：円形 or 矩形のCSSカラーブロック
- 「before / after」比較レイアウトを1箇所（診断前後の変化）

---

## セクション構成
1. **HERO** — ブランド名（Cormorant Garamond 大）+ 「あなたの色を、見つけよう」+ 4色のグラデーション背景
2. **WHAT IS** — パーソナルカラーとは（テキスト主体 + カラースウォッチ装飾）
3. **4 SEASONS** — 春夏秋冬4タイプ（各シーズンカラー背景 + 代表色スウォッチ + 特徴テキスト）
4. **BEFORE / AFTER** — 診断前後の変化（テキストで語る / 引用形式）
5. **SERVICE** — 診断プランと料金（シンプルな料金テーブル / 細線）
6. **BOOK CTA** — 予約へのCTA + FAQ

---

## ボタン・インタラクション（女性向け）
- CTAボタン：4シーズン色でホバーカラーが変わる演出（メインは ink色 → hover でそのページのアクセント色）
- カラースウォッチ：hover で scale(1.15) + 色名がフェードイン（0.3s ease）
- 4シーズンカード：hover で背景が対応シーズン色の薄いバージョンに切り替わる
- セクション遷移：opacity 0 + translateY(12px) → 1 + 0（0.9s ease / stagger）

---

## 必須ディテール（最低2つ）
- [ ] 4色グラデーション：ヒーロー背景を spring / summer / autumn / winter の横グラデーションに（各25%・くすみトーン）
- [ ] カラースウォッチUI：CSS円形ブロックを色見本として多用（border 1px mid / hover で拡大）
- [ ] シーズン別背景切り替え：4シーズンセクションがスクロールで bg-color が変わる
- [ ] before/after テキストアニメ：2カラムが左右から中央に向かってスライドイン

---

## 装飾ルール
**OK：** 極細ボーダー / CSSカラースウォッチ / グラデーション（くすみトーン限定）/ シーズン別カラー
**禁止：** 絵文字 / アイコン / box-shadow blur 8px超 / 均等割り / 高彩度グラデ / ネオン系

---

## 感情設計（コピー指針）
- 「診断」でなく「発見」：「あなたの色を知る」ではなく「あなたに合う世界が広がる」
- 科学的根拠を添えつつ、感覚的な言葉で語る：「骨格と色彩理論に基づく」→「だから、しっくりくる」
- before/afterは数字でなく感情で：「30分の診断後、ショッピングが楽しくなった」
- 失敗の共感から入る：「似合わない服を買い続けてきた、あの感覚」
