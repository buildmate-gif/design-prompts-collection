```
以下の要件に従って、アウトドア用品のプロダクト紹介LPをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- ブランド名：[ブランド名]
- 商品名：[商品名]
- カテゴリ：[例：バックパック / テント / レインウェア]
- キャッチコピー：[一言キャッチ（英語または日本語）]
- 商品説明：[2〜3文。素材・技術・用途の特徴]
- スペック：[重量] / [容量または寸法] / [素材] / [防水規格] / [カラーバリエーション]
- 特徴・テクノロジー：[特徴1：説明] / [特徴2：説明] / [特徴3：説明]
- 使用シーン：[シーン1] / [シーン2] / [シーン3]
- 価格：[¥XX,XXX（税込）]
- CTA文言：[例：購入する / 詳細を見る]
- URL / 連絡先：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Barlow Condensed + Source Serif 4）のみ使用

【世界観・参照スタイル】
- Arc'teryx / MYSTERY RANCH / Black Diamond の製品ページを参照
- 「山岳救助隊の装備リストを、デザイン誌に載せたような厳格な美しさ」
- Mil-Spec Meets Alpine / 機能仕様書の審美性 / Equipment Intelligence

【カラーパレット（厳守）】
- ベース：スレートブルーグレー #2B3A4A / ディープスレート #1E2A36
- サーフェス：シルバーモスト #EEF0F0 / ミッドグレー #BEC3C8
- テキスト（ライト面）：スレート #2B3A4A / グレー #5A6472
- テキスト（ダーク面）：スノー #EEF0F0 / ライトグレー #AAB2BB
- アクセント：1色のみ（ロープタン #C8A96E）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、蛍光色、グラデーション全般

【タイポグラフィ】
- 商品名（H1）：Barlow Condensed / clamp(4rem, 9vw, 8rem) / font-weight: 800 / uppercase / letter-spacing: -0.02em
- スペックラベル：Barlow Condensed / uppercase / 0.65rem / font-weight: 600 / letter-spacing: 0.25em / color: ロープタン
- セクション見出し（H2）：Barlow Condensed / clamp(1.4rem, 3vw, 2.2rem) / font-weight: 600 / uppercase
- 本文・説明：Source Serif 4 / 1rem / line-height: 1.8 / letter-spacing: 0.02em
- 価格・数値：Barlow Condensed / clamp(2rem, 4vw, 3rem) / font-weight: 700 / color: ロープタン
- 禁止：Inter / Roboto / Arial / system-ui

【レイアウト原則】
- 全体は2モードで構成：「ダーク面（スレートブルー背景）」と「ライト面（シルバーモスト背景）」を交互に
- ヒーローは全画面ダーク。商品名を画面左半分に縦長配置、右は商品プレースホルダー（斜め切り抜き風）
- スペック一覧：横スクロール式パネル（PC：5列ゆったり / モバイル：2列で横スクロール）
- 特徴セクション：左右交互配置（ジグザグ）で各テクノロジーを説明
- 使用シーンは縦長の「フィールドログカード」として3枚並列

【ページ構成（この順番で実装）】
1. Hero（ダーク）：商品名（巨大）+ キャッチコピー + ブランド名 + アンカーCTA
2. Specs Ticker：スペック値が流れる横スクロールマーキー（Barlow Condensed / 大型）
3. Technology（ライト）：3つの特徴をジグザグレイアウト
4. Spec Table（ダーク）：縦型スペック比較テーブル（重量・素材・規格を横幅いっぱいに）
5. Field Scenes（ライト）：使用シーン3枚カード（縦長）
6. Pricing + CTA（ダーク）：価格を大きく + CTAボタン（ロープタン）

【装飾・インタラクション】
- ヒーロー商品ビジュアル：マウスのX/Y移動に合わせて最大±6pxのtransform parallax（JS / mousemove）
- スペック数値：スクロール時に0からオドメーター式カウントアップ（Intersection Observer）
  - 例：「0g → 1,240g」を2秒かけてカウント
- 横スクロールマーキー（Specs Ticker）：CSSアニメーションで無限ループ / ホバーで一時停止
- ジグザグ特徴ブロック：スクロールで左右から交互にslide-in（0.6s ease-out）
- CTAボタン：ホバーでロープタン塗り + テキストがスレートに反転 + 右矢印がtranslateX(6px)

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 8px 超（ギア感を維持）
- 彩度の高いグラデーション
- 均等割り3カラムカード
- 紫・青のグラデーション
- カード型のふわふわしたシャドウ

【「忘れられないディテール」（最低2つ必須実装）】
- [ ] **オドメーターカウントアップ**：重量・防水規格数値が0からリールするように更新
- [ ] **商品パラックス**：マウス追従でギアが僅かに3D的に傾く（mousemove / transform）
- [ ] **Specs Ticker**：スペックラベルと値が高速で流れる帯（ロープタン × スノー）
- [ ] **斜め分割ライン**：ダーク→ライトのセクション切替をclip-path: polygon()で斜め分割

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルでは横スクロールマーキーとスペックパネルを縦積みに変換
- ジグザグレイアウトはモバイルで全て縦積みに
```
