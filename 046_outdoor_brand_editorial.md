# F46 — アウトドアブランド エディトリアルマガジン
## 男性向け / ブランドストーリー・フィールドレポートページ

---

## 既存との差分確認（被りなし根拠）

| 項目 | 本プロンプトの選択 | 最も近い既存との違い |
|------|-----------------|-------------------|
| **カラー** | フォレストモス #3D4A35 × バーチクリーム #F0EBD8 × チャコール #2A2A26 × コパー #B87333 | 016のオレンジ系、035のカーキとは明度・彩度・組み合わせが異なる。フォレストモス × コパーは未使用 |
| **フォント** | Fraunces + DM Serif Text | この2本の組み合わせは全44本で未使用（Frauncesは全本未使用） |
| **レイアウト** | 新聞×トポグラフィックマップ風のランダム角度グリッド / pull-quote大型縦配置 | 022のレタープレスは横2段組・欄外配置が主軸。本案はマップ的斜め配置が主軸で異なる |
| **アニメーション** | SVGトポグラフィックライン描画 + テキストの「刻印」スタンプアニメーション | トポグラフィ描画は全44本未使用 |
| **テーマ** | アウトドアブランドのフィールドレポート / 自然環境×男の冒険 エディトリアル | 035は博物館的百科辞典。本案はフィールドで書かれたジャーナリズム |

## 使い方
以下のコードブロック内のプロンプトをそのままコピーして、[　]内のプレースホルダーを差し替えてください。

---

```
以下の要件に従って、アウトドアブランドのエディトリアルマガジンページをHTML+CSS（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━
- ブランド名：[ブランド名]
- 特集タイトル：[例：「荒野に線を引く。」]
- 特集サブタイトル：[1〜2文のリード]
- フィールドレポート：[場所名：エリア説明：体験レポート2〜3文] / [場所2...] / [場所3...]
- 取り上げるギア：[ギア名：カテゴリ：一言コメント] / [ギア2...] / [ギア3...]
- フォトグラファー / ライター名：[名前]（架空可）
- 出版日 / 号数：[YYYY年MM月 / Vol.XX]
- 購読・問い合わせ：[URL またはメール]

━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- HTML + CSS + JavaScript（単一ファイル）
- 外部フォント：Google Fonts（Fraunces + DM Serif Text）のみ使用

【世界観・参照スタイル】
- Monocle × Alpinist Magazine × Patagonia Environmental Grants誌面を参照
- 「テント場の夜、ヘッドライトで読む。紙の匂いがするウェブページ」
- Expedition Journalism / Topographic Aesthetics / フィールドで読まれるものの密度

【カラーパレット（厳守）】
- ベース（ライト面）：バーチクリーム #F0EBD8 / ウォームペーパー #E6DFC8
- ベース（ダーク面）：フォレストチャコール #2A2A26 / ディープモス #1E261A
- テキスト（ライト面）：チャコールインク #2A2A26 / ダークモス #3D4A35
- テキスト（ダーク面）：バーチクリーム #F0EBD8 / ライトタン #C4B89A
- アクセント：1色のみ（コパーブラウン #B87333）
- 禁止：純白 #FFFFFF、純黒 #000000、原色、派手なグラデーション

【タイポグラフィ】
- 特集タイトル（H1）：Fraunces / clamp(3rem, 7vw, 6.5rem) / font-weight: 900 / font-style: italic / letter-spacing: -0.03em
- セクション見出し（H2）：Fraunces / clamp(1.6rem, 3vw, 2.5rem) / font-weight: 700
- プルクオート：Fraunces italic / clamp(1.8rem, 3.5vw, 2.8rem) / font-weight: 400 / color: コパーブラウン
- 本文：DM Serif Text / 1.05rem / line-height: 2.0 / letter-spacing: 0.015em
- フィールドデータ（高度・気温・日数）：Fraunces / uppercase / 0.72rem / letter-spacing: 0.2em / color: コパーブラウン
- 写真キャプション：DM Serif Text italic / 0.85rem / color: ダークモス or ライトタン
- 禁止：Inter / Roboto / Arial / system-ui

【レイアウト原則】
- 全体を「トポグラフィックマップを広げた机」に見立てたレイアウト
- 記事テキストの最大幅：700px / 中央配置
- 特集タイトルは本文幅を超えてはみ出す（130〜150%幅）
- プルクオートは本文の左外にはみ出す縦長配置（writing-mode: horizontal-lr を維持しつつ左にネガティブマージン）
- フィールドデータ帯：フルブリードのコパーブラウン帯、テキスト色はバーチクリーム
- ギア紹介：本文の右に浮かぶ「サイドバー」スタイル（float または sticky-ish）
- セクション切替：clip-path:polygon()で斜め分割（ライト面→ダーク面）

【ページ構成（この順番で実装）】
1. Masthead（ダーク）：ブランド名 + 号数 + 日付 / 細いコパーラインで区切り
2. Cover Title（ライト）：特集タイトル（はみ出し配置）+ リード文 + フォトグラファー名
3. Field Data Ticker（コパー帯）：高度 / 気温 / 行動日数 / 累積標高がマーキーで流れる
4. Report 1（ライト）：フィールドレポート1（本文 + プルクオート + サイドバーギア）
5. Report 2（ダーク）：フィールドレポート2（セリフ本文 + 大型プルクオート）
6. Gear Lineup（ライト）：取り上げたギア一覧（横型カード、斜め区切り）
7. Colophon（ダーク）：購読案内 + ブランドステートメント

【装飾・インタラクション】
- Masthead→Cover Titleの遷移：SVGトポグラフィックライン（等高線）がStroke-Dasharrayで描画されながら登場
  - 等高線はCSSのconic-gradient風の同心楕円をSVGで実装
- Field Data Ticker：CSSアニメーションで左方向に無限ループ / ホバーで停止
- プルクオート登場：Intersection Observer で画面に入ったとき左から translateX(-30px)→0 にslide-in（1s ease）
- テキストスタンプ効果：フィールドデータラベル（ELEV / TEMP / DAYS）がページロード時に
  opacity:0 → 1 + letter-spacing: 0.5em → 0.2em の「刻印」アニメーション（stagger 0.15s）
- スクロール読了バー：上部2pxのコパーブラウンラインがscrollTopに連動して伸びる

【完全禁止（厳守）】
- 絵文字・アイコン（FontAwesome含む）一切禁止
- box-shadow の blur 半径 8px 超
- border-radius 10px 超（自然素材感を維持）
- 彩度の高いグラデーション
- 均等割り3カラムカード
- 紫・青のグラデーション
- ボックスシャドウによる「浮いたカード」感

【「忘れられないディテール」（最低2つ必須実装）】
- [ ] **トポグラフィックライン描画**：SVGの等高線がstroke-dasharrayで徐々に描かれるアニメーション
- [ ] **スタンプ刻印アニメーション**：フィールドデータラベルが letter-spacing 収縮で「刻まれる」演出
- [ ] **Field Data Ticker**：高度・気温などのデータがコパー帯をループするマーキー
- [ ] **プルクオート縦長配置**：Frauncesのイタリック体がページ左に大きく存在する「声」

【レスポンシブ】
- モバイル（375px〜）/ タブレット（768px〜）/ デスクトップ（1280px〜）
- モバイルではサイドバーギアを本文下に縦積み
- プルクオートのはみ出しはモバイルで解除し、本文幅内に収める
- トポグラフィックSVGはモバイルで縮小表示（100vw基準）
```
