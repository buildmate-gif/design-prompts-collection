## 既存50本との差別化チェック

| 項目 | 既存の使用状況 | 本プロンプトの選択 | 重複なし |
|------|--------------|-----------------|---------|
| **カラー** | パステル系は全50本で禁止または未使用。明るいスカイブルー×イエロー×コーラル3色構成は未登場 | スカイブルー #5BB8F5 × サニーイエロー #FFD85C × コーラル #FF7E5F | ✅ |

```
以下の要件に従って、学習塾のホームページをHTML+CSS+JavaScript（単一ファイル）で作成してください。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ コンテンツ情報（必ず差し替えてください）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 塾名：[塾の名前]
- キャッチコピー：[メインキャッチコピー（例：「まなぶって、たのしい！」）]
- サブコピー：[2〜3文。対象学年・強み・雰囲気の説明]
- 特徴3点：[特徴1] / [特徴2] / [特徴3]
- 対象学年：[例：小学1年生〜中学3年生]
- 開講科目：[例：国語・算数・英語・理科・社会]
- 体験授業CTA：[ボタンテキスト（例：「無料体験に申し込む」）]
- CTAリンク先URL：[URL または #]
- 住所・電話番号：[任意]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ ページ構成（セクション定義）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
以下の順序でセクションを構築すること。

1. HEADER（固定ナビ）
   - 背景：スカイブルー #5BB8F5
   - 塾名を左側に Baloo 2 / bold / 白 #FFFFFF で表示
   - 右側にナビリンク（ひらがな表記推奨：「とくちょう」「かもく」「たいけん」）
   - ナビリンクは pill 形状（border-radius: 20px）、hover時に背景が白 #FFFFFF に変化・文字色が #5BB8F5 に

2. HERO（絵本の表紙風）
   - 背景色：サニーイエロー #FFD85C
   - ページ中央に大きな「本の形」SVGフレーム（幅700px・角丸20px・白背景・影あり）を描く
   - フレーム内の左半分：カラフルな学用品のSVGイラスト（鉛筆・本・虫眼鏡・星をシンプルな図形で描く）
   - フレーム内の右半分：H1キャッチコピー + サブコピー + CTAボタン
   - H1：Baloo 2 / 3.5rem / コーラル #FF7E5F / font-weight: 800
   - CTAボタン：コーラル #FF7E5F 背景・白テキスト・border-radius: 50px・padding: 1rem 2.5rem
   - フレーム全体に CSS keyframes で上下に漂うアニメーション（floating）を適用
     （transform: translateY(-8px) ↔ translateY(8px)、duration: 3s、ease-in-out、infinite）

3. FEATURES（特徴3点：見開き絵本交互レイアウト）
   - セクション背景：オフホワイト #FAFAF8
   - 3つの特徴を「見開き本の左右ページ」スタイルで表示
   - 奇数項目：左側に丸いイラスト枠（背景色 #5BB8F5）、右側にテキスト
   - 偶数項目：左側にテキスト、右側に丸いイラスト枠（背景色 #FFD85C）
   - 各特徴の見出しは Baloo 2 / 1.8rem / コーラル #FF7E5F
   - 各ブロック間にはページの区切りを示す点線（border-top: 2px dashed #C8E8FF）

4. SUBJECTS（開講科目：カラフルカードグリッド）
   - セクション背景：スカイブルー #5BB8F5
   - 科目ごとに吹き出し型カードを表示（CSS clip-pathで下部に三角を作る）
   - カード背景色はローテーション（#FFFFFF / #FFD85C / #FF7E5F / #AADEA7 / #C9A8E2）
   - 各カードにその科目を連想する1文字SVGアイコン（✏ 📐 🔤 等ではなくSVGで描く）
   - 科目名：Baloo 2 / 1.2rem / bold / #333333
   - カードに hover時にpop-up（scale: 1.08、transition: 0.2s）

5. CTA（体験授業申し込み）
   - セクション背景：コーラル #FF7E5F
   - 中央に白い大きなカード（border-radius: 24px・padding: 4rem）
   - カード内：「まずは無料体験から！」（Baloo 2 / 2.5rem / コーラル）+ サブテキスト + 大きなCTAボタン
   - ボタン：コーラル #FF7E5F 背景・白テキスト・border: 3px solid #FFFFFF・border-radius: 50px
   - カード四隅に小さな星形SVG（#FFD85C）を配置（装飾）
   - ボタンに bounce アニメーション（keyframes: 0%→100% translateY(0)、50% translateY(-6px)、duration: 1.5s、infinite）

6. FOOTER
   - 背景：#333333
   - 塾名（白・Baloo 2）＋住所・電話番号（#AAAAAA・Noto Sans JP）
   - 上部にカラフルな点線ボーダー（repeating-linear-gradient で #5BB8F5・#FFD85C・#FF7E5F を繰り返す）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ デザイン要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【出力形式】
- 単一HTMLファイル（style・scriptタグをhead内またはbody末尾にインライン記述）
- 外部リソース：Google Fonts（Baloo 2 + Noto Sans JP）のみ許可
- 外部CSSライブラリ・UIフレームワーク一切禁止
- 画像ファイル一切使用禁止（すべてCSS・SVGで表現）

【世界観】
- 絵本の世界に迷い込んだような、丸みと温かみのある子ども向けデザイン
- 「勉強が楽しくなる場所」を視覚で伝える
- キーワード：Storybook / Round & Warm / Colorful Adventure / Safe Space

【カラーパレット（HEXコードを厳守）】
- スカイブルー：#5BB8F5（メインカラー：ヘッダー・背景）
- サニーイエロー：#FFD85C（HEROセクション・アクセント）
- コーラル：#FF7E5F（CTA・見出し・ボタン）
- ソフトグリーン：#AADEA7（科目カードのアクセント）
- ソフトパープル：#C9A8E2（科目カードのアクセント）
- オフホワイト：#FAFAF8（特徴セクション背景）
- テキスト標準：#333333
- テキストサブ：#666666
- 禁止：純白 #FFFFFF（カード背景のみ可）、純黒 #000000

【フォント設定】
- メイン見出し・装飾テキスト：Baloo 2（英数字・ひらがな見出しに適用）
- 本文・日本語テキスト：Noto Sans JP
- 日本語本文：font-size 1rem / line-height 1.9 / letter-spacing 0.02em

【形状ルール】
- border-radius は積極的に使用（最小8px・最大50px / pill形状）
- 直角（border-radius: 0）は一切禁止
- box-shadow：0 4px 20px rgba(0,0,0,0.08) を標準として使用

【アニメーション（2種のみ使用）】
1. floating：HEROフレームに適用。transform translateY -8px〜8px、3s、ease-in-out、infinite
2. bounce：CTAボタンに適用。translateY 0〜-6px〜0、1.5s、ease-in-out、infinite

【レスポンシブ】
- ブレークポイント：768px
- モバイル時：見開きレイアウトを縦積みに変更、H1は clamp(2rem, 8vw, 3.5rem)
- ナビはハンバーガーメニュー不要（ロゴのみ表示でOK）

【禁止事項】
- 絵文字（emoji）の使用禁止（SVGで代替）
- グリッチ・点滅・過度なパーティクル禁止
- box-shadow blur 20px超 禁止
- 背景 gradient の多用禁止（セクション単色ベタ塗り推奨）
```
