## 目的

地方移住を検討している都市生活者をターゲットにした、自治体または移住コンシェルジュサービスのLP。
「移住＝不便・諦め」という先入観を静かに覆し、
「都市では手に入らない豊かさ」への好奇心と具体的な行動を促す。
情報量が多くなりがちな自治体サイトとは一線を画す、
**感情から入って理性が追いかけてくる**設計にする。

---

## フォント

- 大見出し（キャッチ）：**DM Serif Display** / weight 400 / size clamp(2.8rem, 6vw, 5.5rem) / letter-spacing 0.02em
  - Italic は使用しない（力強く、土地に根差した印象）
- 地名・セクションラベル：**Instrument Serif** / weight 400 / size clamp(1rem, 2vw, 1.6rem) / letter-spacing 0.06em
- 本文：**Noto Sans JP** / weight 300 / size 0.9rem / line-height 2.3 / letter-spacing 0.05em
- 数字・統計データ：DM Serif Display / size clamp(3rem, 7vw, 6rem) / weight 400
- **禁止：Inter / Roboto / Arial / system-ui / Noto Serif JP（和文セリフは使わない）**

## カラーパレット

```css
--bg:         #EEF2EC   /* ペールモス：山の朝靄 */
--bg2:        #E4EBE0   /* ライトフォレスト：林の透過光 */
--bg-dark:    #1C3028   /* ミッドナイトフォレスト：深い森 */
--ink:        #1A2420   /* ディープモス：濃い墨緑 */
--mid:        #627A6C   /* スレートグリーン：苔むした岩 */
--terra:      #C85A2A   /* テラコッタオレンジ：土の色・CTAに */
--mist:       #B8C8B4   /* ミストグリーン：霧・アクセントライン */
--cream:      #F4F0E8   /* クリームホワイト：紙・余白感 */
```

純白・純黒・蛍光系・彩度の高いグラデーション 禁止。
「清流・棚田・朝靄・夕暮れの山並み」を想起させる、落ち着いたアースカラー体系。

---

## レイアウト

- 12カラムグリッド（均等割り禁止）
- section padding 10rem〜14rem（mobile 5rem）
- **斜線グリッドの导入**：一部セクションで `clip-path: polygon(0 0, 100% 5%, 100% 95%, 0 100%)` を用いた台形断面の帯
- 地形・地図感を意識した**有機的な非対称配置**：テキストと画像の比率を 3:7 / 7:3 で交互に
- フルブリードのダーク（bg-dark）セクションを1箇所（深夜の山村の静けさ）
- 「風景の切り抜き」：画像プレースホルダーを `aspect-ratio: 4/3` と `aspect-ratio: 2/3` の混在で

## セクション構成

1. **MOVE** — ヒーロー：DM Serif Display大見出し + 「あなたが探していたのは、もしかして"ここ"かもしれない」+ 台形断面の背景帯
2. **THIS TOWN** — 地域紹介：地名（Instrument Serif）+ 3つのキーワード + 非均等画像グリッド
3. **LIFE** — 暮らしの4シーン（農・食・仕事・子育て）横長カードを交互配置
4. **DATA** — 移住サポート実績数値（フルブリードダーク背景 / DM Serif Display大数字）
5. **VOICE** — 移住者の声：3〜4名（引用形式 / 出身地・移住先・年代 付き）
6. **STEP** — 移住相談から引越しまでの5ステップ（横型タイムライン）
7. **CONSULT CTA** — 「まず話を聞かせてください」テラコッタCTA + よくある不安Q&A

---

## ボタン・インタラクション

- CTAボタン：テラコッタ背景 / 角丸なし / hover で bg-dark に反転（0.4s ease）
- 地域カード：hover で画像が `scale(1.03)` + overlay に地域キーワードが opacity 0→1
- ステップ横タイムライン：スクロールで左から順に `translateX(-20px)→0` + opacity（stagger 0.15s）
- 移住者の声：スクロールで `translateY(12px)→0` + opacity（1s ease）

---

## 必須ディテール（最低2つ）

- [ ] **clip-path カーテンリビール**：セクションの背景帯が `clip-path: polygon(0 8%, 100% 0, 100% 92%, 0 100%)` の台形に切り抜かれ、スクロールで徐々に展開する（JS + CSS transition）
- [ ] **数字カウントアップ**：DATA セクションの実績数値が viewport に入ると 0 から目標値まで 1.5s でカウント
- [ ] **地形ライン**：ヒーロー背景に SVG の緩やかな稜線（山のシルエット）を極薄（opacity 0.08）で配置
- [ ] **ダーク切り替え**：DATA セクションで bg が bg-dark に切り替わる際、境界線が台形（clip-path）でつながる

## 装飾ルール

**OK：** clip-path 台形断面 / 極細ボーダー（1px mist） / SVG稜線ライン / backdrop-filter blur(8px) / SVG noise opacity 0.04
**禁止：** 絵文字 / アイコン / box-shadow blur 8px超 / 均等割り3カラムカード / 派手グラデーション / border-radius 12px超（台形を除く）

## 感情設計（コピー指針）

- 「移住しませんか」ではなく「あなたの理想の暮らし、もう実在している」
- 不安を認める：「慣れた街を離れる怖さ、わかります」→「だから一緒に考える」
- 数字より感情：「人口3,000人の町」より「夜、星が手で掴めそうなくらい近い」
- 「諦め」でなく「選択」：移住は降格でなく、もっと豊かな人生への転換
- ターゲットの具体的なシーン：「電車を降りたとき、なぜかほっとした」

## 補足

- ブランド名：**IRODORI**（移住コンシェルジュサービス / 架空）
