## 目的

創業100年を超える地方の老舗旅館のブランドサイト。
地元住民には「誇り」を、観光客には「ここにしかない体験」を伝える。
大手予約サイトに並ぶ「施設情報ページ」ではなく、
**この宿の精神と、積み重ねてきた時間を語るブランドのページ**。
「泊まりに行く」という行動より先に「この宿を知りたい」という引力を生む。

---

## フォント

- 宿名・大見出し：**Kaisei Decol** / weight 400 / size clamp(2.5rem, 6vw, 5.5rem) / letter-spacing 0.15em
  - 明朝系でありながら現代的な清潔感を持つフォント。筆の払いが美しい。
- 英字アクセント・引用：**Spectral** / Italic / weight 300 / size clamp(1rem, 2vw, 1.8rem) / letter-spacing 0.05em
- 本文：**Noto Serif JP** / weight 400 / size 0.92rem / line-height 2.4 / letter-spacing 0.07em
- 数字・年号（創業年等）：Kaisei Decol / size clamp(4rem, 10vw, 9rem) / weight 400 / opacity 0.06（背景的に大きく）
- ラベル・セクション番号：Spectral / uppercase / 0.62rem / letter-spacing 0.28em
- **禁止：Inter / Roboto / Arial / system-ui / Noto Sans JP（サンセリフは使わない）**

## カラーパレット

```css
--bg:          #F6F2EA   /* 花弁白：障子越しの光 */
--bg2:         #EDE8DC   /* 生成り：和紙の温もり */
--bg-season:   #E8F0E4   /* 若竹：春の茶室 */
--ink:         #1A1810   /* 墨：最も濃い色 */
--mid:         #7A7268   /* 石畳：中間調 */
--bamboo:      #4A6848   /* 老竹：深い緑・控えめなアクセント */
--sand:        #D4C4A0   /* 砂：縁側の光 */
--engawa:      #B8A88C   /* 縁側：セクションボーダーに */
```

純白・純黒・蛍光色・高彩度グラデーション 禁止。
「光の差し込む客室・苔むした中庭・水屋の静けさ」を連想させるトーン。
老竹（bamboo）はアクセントとして最大8回まで使用。

## レイアウト

- **横スクロールスナップ**：「廊下」セクションを1箇所実装（CSS scroll-snap-type: x mandatory）
  - 客室を横に並べ、スクロールで1部屋ずつ「扉を開ける」ような体験
- 12カラムグリッド（均等割り禁止）
- section padding 12rem〜16rem（mobile 6rem）——「宿の間合い」の余白
- **縦書き（writing-mode: vertical-rl）**を2〜3箇所：セクション左端または右端に
- 1px の極細ラインを「建具の細い框」として横断的に使用
- 画像プレースホルダー：縦長 3:4 / 正方形 1:1 / 横長 16:9 の3種混在
- テキストと画像の重なり（overlap）を2箇所以上

## セクション構成

1. **暖簾（NOREN）** — ヒーロー：宿名（Kaisei Decol 大）+ 「ここに来るたびに、少し正直になれる気がする」+ 縦書きセクションラベル + フルスクリーン画像
2. **此処（KOKO）** — 宿の紹介：創業年（背景的大数字）+ 理念文（Noto Serif JP / 広い行間）+ 縦書き章タイトル
3. **廊下（ROKA）** — 客室横スクロールスナップ：3室を横並び（部屋名 / 特徴 / Spectral英訳）
4. **食（SHOKU）** — 料理紹介：縦長画像 + 食材へのこだわり文 + 季節の器の説明
5. **時間（JIKAN）** — 1日のタイムライン：朝〜夜の宿での過ごし方（縦型・点線ライン）
6. **声（KOE）** — 宿泊客の声：2〜3件（引用形式 / 宿泊年 + 出身地のみ）
7. **予約（YOYAKU）** — 老竹背景 / 「初めてのご予約・お問い合わせはこちらから」CTA

---

## ボタン・インタラクション

- CTAボタン：bamboo 1px border / 背景なし → hover で bamboo が右から fill（clip-path animate 0.5s ease）
- 客室カード（横スクロール）：hover で画像が `sepia(0.15)→sepia(0)` + 部屋名がフェードアップ
- 縦書きラベル：スクロールで `translateY(20px)→0` + opacity（1.2s ease）
- タイムライン点線：IntersectionObserver で上から伸びるアニメーション（`scaleY: 0→1` / transform-origin: top）

---

## 必須ディテール（最低2つ）

- [ ] **暖簾マスクリビール**：ヒーロー画像が上から下へ `clip-path: inset(0 0 100% 0)` → `inset(0 0 0% 0)` でゆっくり現れる（1.8s cubic-bezier(0.16, 1, 0.3, 1)）——暖簾をくぐる瞬間の感覚
- [ ] **横スクロールスナップ廊下**：客室を `overflow-x: scroll; scroll-snap-type: x mandatory` で横並び。各室が `scroll-snap-align: start` で一室ずつパチンと止まる
- [ ] **大型年号背景**：「此処」セクションに創業年（例：一九二三）を Kaisei Decol / `font-size: clamp(6rem, 18vw, 16rem)` / opacity 0.05 で背景的に配置
- [ ] **縦書き季節語**：右端または左端に `writing-mode: vertical-rl` で季節の言葉（例：「秋晴れの候」）を Kaisei Decol / letter-spacing 0.2em で静かに配置

## 感情設計（コピー指針）

- 「施設案内」を「物語の入口」に変える：部屋名は機能でなく情景
- 「贅沢・豪華」より「静かで深い」：宿の格は「静けさ」で表現
- 地元への敬意：「この土地の時間に、少し混ぜてもらう旅」
- 予約導線は命令形にしない：「泊まってみませんか」より「もし気が向いたら、のれんをくぐりにきてください」
- 年号・歴史を「重さ」ではなく「信頼」として：「100年、ただ同じことを続けてきた」

## 補足

- 宿名：**旅籠 結（YUKI）** / 架空の老舗旅館
- 所在地：架空（例：山形県某村 / 京都府某町）
- 創業年：架空（例：大正12年 / 1923年）
- コピー：日本語主体（セクションラベルと引用テキストのみ英語）
- 画像：CSSプレースホルダー（墨〜砂グラデーション / bg-dark 〜 bg2 トーン）
- 料金表記：あり（シンプルな1行「一泊二食 ¥〇〇,〇〇〇〜」のみ）
- SNS・口コミサイトへのリンク：一切不要
- フッター：宿名 + 所在地 + 電話番号 + 「暖簾をくぐるような気持ちでお電話ください」の一文
