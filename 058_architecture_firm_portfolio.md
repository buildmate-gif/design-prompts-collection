## 目的

建築設計事務所（住宅・商業・リノベーション）のポートフォリオ兼サービスサイト。
Tadao Ando / SANAA / 隈研吾 的な「余白が語る建築の哲学」を、Webで体現する。

設計図面の精度と完成物の詩情を両立。「この事務所に頼みたい」という引力を生む。

建設業界DXに深く関わるBuildMate的文脈からも実用性の高いサンプル。

---

## フォント

```
- ブランド名・H1：Cormorant / weight 300 / size clamp(3rem, 6vw, 6rem)
  letter-spacing: -0.01em（セリフ体の芸術的な重み）
  em { font-style: italic; color: var(--rust); }
- セクション見出し・数値・スペック：Archivo Narrow / weight 500–700
  size varies / letter-spacing: 0.2em / text-transform: uppercase
- 本文（日本語）：Noto Sans JP / weight 300 / size 0.84–0.88rem / line-height 2.2
- サービス番号：Archivo Narrow / weight 700 / size 0.65rem / letter-spacing 0.2em
```

**禁止：Inter / Roboto / Syne / Bebas Neue / system-ui**

## レイアウト

- ヒーロー：2カラム（左：コンテンツ、右：フルハイト画像）/ fixed-header（mix-blend-mode）
- 作品集：CSS grid-template-areas による間取り図的不規則グリッド（5作品）
- フィロソフィー：2カラム / 画像を2枚重ねた複合配置
- サービス：ダーク背景 × 4カラムグリッド / 上ボーダーホバー
- プロセス：縦リスト / 大型番号（2.5rem / opacity 0.2）
- 事務所について：2カラム（画像 + テキスト + プロフィールグリッド）

## セクション構成

1. **HERO** — 設計事務所らしい2カラム / 「空間が語る、設計の意志。」
2. **SELECTED WORKS** — 不規則グリッドの作品集（ホバーでタイトル出現）
3. **PHILOSOPHY** — 「引き算の設計」の哲学（2カラム）
4. **SERVICES** — 4種のサービス（住宅・集合住宅・商業・リノベーション）
5. **PROCESS** — 設計の進め方（6ステップ）
6. **ABOUT** — 代表プロフィール + 事務所情報
7. **CTA** — ブループレートの接触セクション
8. **FOOTER** — ロゴ + 実績 + 連絡先
